# FutureTense L3: «Не буду и не смогу» — отрицательное будущее (არ / ვერ)

**Задача:** ROADMAP.md → задача #62 · GitHub issue #409  
**Статус:** ready

---

## Контекст проблемы

После FutureTense L1–L2 ученик умеет сказать «я напишу» (დავწერ) и «он съест» (შეჭამს), но не может:
- Отказать: «не буду» / «не хочу делать»
- Признать невозможность: «не смогу» / «не в состоянии»

Это базовые коммуникативные нужды туриста (отказаться от блюда, объяснить, что не успеешь). Без них модуль «Будущее время» функционально неполон: ученик может говорить о планах, но не об отсутствии планов.

---

## Обучающий элемент

### Главный reveal-момент: «форма не меняется — меняется только частица»

Ученик уже знает и-ვ-делают будущее:

```
ვ-წერ  →  და-ვ-წერ
пишу      напишу
```

Открытие: **будущая форма остаётся той же, перед ней просто встаёт частица**:

```
და-ვ-წერ  →  არ  და-ვ-წერ   (не буду писать)
              ვერ და-ვ-წერ   (не смогу написать)
```

Никакой перестройки глагола. Ученик осознаёт: «Я уже знаю всё нужное для будущего — мне нужны только два слова, чтобы добавить отрицание».

### Вторичный обучающий момент: «არ» vs «ვერ» — это разница «не хочу» vs «не могу»

Два слова — два разных смысла. Эта разница есть в любом языке, но в грузинском она морфологически закреплена:

| Частица | Значение | Аналогия в русском |
|---------|----------|---------------------|
| **არ** | отказ, нежелание | «не буду» (сам решил) |
| **ვერ** | невозможность | «не смогу» (внешнее / физическое ограничение) |

**«Wow»-момент:** спутать «არ» и «ვერ» — это как перепутать «не хочу» и «не могу» в ответ на вопрос «почему не пришёл». Разница вежливости и честности.

#### Блок 2 — NegationDuo (новый компонент, см. ниже)

Две плитки рядом, демонстрирующие обе частицы:

```
┌──────────────────────┐  ┌──────────────────────┐
│    არ                │  │    ვერ               │
│                      │  │                      │
│  не буду             │  │  не смогу            │
│  (отказ, нежелание)  │  │  (невозможность)     │
└──────────────────────┘  └──────────────────────┘
```

Левая плитка: `jewel-tile` с ruby (#E01A3C) бордером.  
Правая плитка: `jewel-tile` с navy (#1B5FB0) бордером.  
Georgian символ: T1 Manrope 800, jewelInk.  
Подпись: T4 Manrope 600.  
Описание: T5 Manrope 400, jewelInk с 70% opacity.

Семантика цветов:
- **ruby** → «არ»: активный выбор, сигнал «стоп», отказ — ruby как «красная карточка»
- **navy** → «ვერ»: внешнее ограничение, спокойная невозможность — navy как «холодный барьер»

#### Блок 4 — List (разбор различия)

- **არ** — ты сам так решил. «Не хочу» / «откажусь» / «не собираюсь».  
  _Пример: «Нет, завтра не буду читать» — твой выбор._
- **ვერ** — ты не можешь, независимо от желания. Физическое, временное или внешнее ограничение.  
  _Пример: «Не смогу прийти — нет времени» — ограничение извне._

---

#### Блок 5 — List (четыре примера из L2)

Те же глаголы, что ученик видел в L2 — теперь в отрицании. Узнавание снижает когнитивную нагрузку:

- `არ დავწერ` — не буду писать (отказ)
- `ვერ შევჭამ` — не смогу съесть (болит зуб, аллергия)
- `არ წავიკითხავ` — не буду читать (не хочу)
- `ვერ გავაკეთებ` — не смогу сделать (нет времени)

---

#### Блок 6 — Paragraph (culture note)

> **Заметка:** В разговоре «ვერ» звучит мягче и вежливее — это признание ограничения, а не нежелания. «არ» честнее, но может прозвучать резче. Как в русском: «не смогу» вежливее, чем «не хочу».

Стиль: note-block (cream фон, gold левый бордер 3px, T5 Manrope 400, italics запрещены — акцент через gold и отступ).

#### Вопрос ft3_q01 — различение контекстом

**Вопрос:** Зуб болит, и ты физически не можешь жевать. Как сказать по-грузински?

**Варианты:**
1. `ვერ შევჭამ` ✓
2. `არ შევჭამ`
3. `შევჭამ`
4. `ვერ ვჭამ`

**Объяснение ✓:** «Болит зуб — это физическая невозможность → **ვერ**. შევჭამ — будущее от «есть» (шевер-б→შე-ვ-ჭამ). Вариант 4 — настоящее время, а не будущее.»

**Объяснение ✗ (для "არ შევჭამ"):** «**არ** означало бы, что ты сам решил не есть. Но раз болит зуб — это невозможность, а не выбор → нужен **ვერ**.»

---

#### Вопрос ft3_q02 — перевод (выбор частицы)

**Вопрос:** Переведи: «Завтра я не буду писать» (это твоё решение, не принуждение)

**Варианты:**
1. `ხვალ არ დავწერ` ✓
2. `ხვალ ვერ დავწერ`
3. `ხვალ დავწერ`
4. `არ ვწერ`

**Объяснение ✓:** «Собственное решение → **არ**. ხვალ = завтра, დავწერ — 1л.ед. будущего от «писать». Вариант 4 — настоящее время (ვწერ), без преверба нет будущего.»

---

#### Вопрос ft3_q03 — перевод грузинского на русский

**Вопрос:** Переведи: «ხვალ ვერ გავაკეთებ»

**Варианты:**
1. `Завтра не смогу сделать` ✓
2. `Завтра не буду делать`
3. `Завтра сделаю`
4. `Уже не сделал`

**Объяснение ✓:** «**ვერ** = невозможность («не смогу»), не отказ («не буду»). გავაკეთებ — будущее от «делать» (გა-ვ-ა-კეთ-ებ). ხვალ = завтра.»

## Компонент: NegationDuo

**Назначение:** парное отображение двух противопоставленных грузинских частиц с цветовой семантикой. Переиспользуется в будущем для других оппозиций (например, «ნუ» vs «არ» в императиве).

**Props:**
```typescript
interface NegationDuoProps {
  left: {
    particle: string;      // «არ»
    label: string;         // «не буду»
    description: string;   // «отказ, нежелание»
    accent: 'ruby' | 'navy' | 'gold';
  };
  right: {
    particle: string;      // «ვერ»
    label: string;         // «не смогу»
    description: string;   // «невозможность»
    accent: 'ruby' | 'navy' | 'gold';
  };
}
```

**Layout:** flex row, gap-3, обе плитки равной ширины (flex-1).  
**Минимальная высота плитки:** 96px (tap target соблюдён, плитки не интерактивны).  
**На 375px:** плитки сохраняют row-layout, текст переносится, минимальный размер шрифта T5 (12px).

**Minankari-токены:**
- Бордер плитки: `1.5px solid jewelInk` с акцентным inner highlight (hairline 0.5px accent-color)
- Offset shadow: `3px 3px 0 jewelInk`
- Фон: `cream`

## Интеграция в ContentProvider

**В `MiniAppContentProvider.cs`**, метод `BuildFutureTenseModule()`:

```csharp
Lesson(3, "«Не буду и не смогу»", "არ / ვერ + будущее", "Отрицательное будущее", "Отрицать планы и признавать невозможность.",
    new List<TheoryBlockDto>
    {
        Paragraph("Ты знаешь «და-ვ-წერ» — я напишу. Частица перед глаголом добавляет отрицание. Форма глагола не меняется."),
        // NegationDuo → два Paragraph с пометкой [ruby] и [navy]
        Paragraph("[ruby]არ[/ruby] — не буду (отказ, нежелание). [navy]ვერ[/navy] — не смогу (невозможность)."),
        List(
            "არ დავწერ — я не буду писать (решение)",
            "ვერ დავწერ — я не смогу написать (ограничение)",
            "არ წავიკითხავ — не буду читать",
            "ვერ შევჭამ — не смогу съесть"
        ),
        Paragraph("Подсказка: «ვერ» звучит мягче в разговоре. «არ» — честнее, но прямее.")
    })
```

*Примечание для Developer: если рендер теории не поддерживает цветовые теги — использовать `NegationDuoBlock` отдельным типом `TheoryBlockDto`. Дизайн этого блока описан в компоненте `NegationDuo` выше.*
