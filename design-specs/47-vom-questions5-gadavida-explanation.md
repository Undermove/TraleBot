# VoM questions5.json m5-q011: исправить объяснение «გადავიდა»

**Задача:** ROADMAP.md → задача #47 · GitHub issue #319  
**Статус:** ready

---

## Контекст проблемы

В `GeorgianVerbsOfMovement/questions5.json`, вопрос `m5-q011`, поле `explanation` ошибочно утверждает:

> «გადავიდა — прош. **1л.ед.** (я перешёл) / 3л.ед. (он перешёл)»

Это фактическая ошибка. Форма «გადავიდა» является **исключительно 3-м лицом единственного числа** аориста. Форма 1-го лица — «**გადავედი**».

Путаница возникает из-за наличия «და-» в начале: ученик видит да-ვ-иდа и по аналогии с настоящим временем (ვ- = 1л.) думает, что это 1л. — тогда как в аористе VoM-глаголов -ვ- здесь часть основы, а не личного маркера.

---

## Обучающий элемент

Не применимо — это исправление фактической ошибки в explanation, а не новая обучающая механика. Корректное объяснение само по себе обучает: ученик узнаёт, что «გადავიდა» и «გადავედი» — разные формы для разных лиц, и не смешивает их при построении речи.

## Что меняется

### Файл: `src/Trale/Lessons/GeorgianVerbsOfMovement/questions5.json`

**Вопрос:** `m5-q011`  
**Поле:** `explanation`

**Было:**
```
«გადავიდა — прош. 1л.ед. (я перешёл) / 3л.ед. (он перешёл)»
```
*(или аналогичная формулировка, приписывающая 1л. форме გადავიდა)*

**Стало:**
```
«გადავიდა — прош. 3л.ед. (он / она перешёл / перешла).
 Форма 1л.ед. — გადავედი (я перешёл).»
```

Ключевые требования к новому тексту:
1. Явно указать, что «გადავიდა» = **3л.ед.** (и только 3л.ед.)
2. Назвать правильную форму 1л.ед.: «**გადავედი**»
3. Не приписывать «გადავიდა» значение «я перешёл»
