**Задача:** ROADMAP.md → задача 11
**Статус:** ready

## Контекст и мотивация

Сейчас для beginners секции грамматики/лексики/продвинутого показываются свёрнутыми с кнопкой «показать все →» — один тап открывает всё сразу. Это не создаёт прогрессивное ощущение пути и убивает мотивацию «разблокировать следующее».

Цель: настоящий прогрессивный путь — следующий раздел физически скрыт до завершения предыдущего. Для intermediate (и для тех, кто явно включил просмотр всего) — всё открыто.

**Существующий код (Dashboard.tsx):**
- `showAllSections` state + `isBeginner` flag уже есть
- Логика collapse секций для beginners уже есть, но показывает locked секции как кнопку «показать все»
- Нужно заменить эту логику на: показ locked-tile вместо кнопки

## Обучающий элемент

**Что учит?** Грузинские слова для каждого раздела:
- `გრამატიკა` (grammar) — пользователь видит его как label заблокированного раздела
- `ლექსიკა` (lexicon) — аналогично
- `გაღრმავება` (deepening / advanced) — аналогично

**Reveal-момент:** Когда раздел разблокируется — на секунду крупно показывается его грузинское название с переводом. Пользователь видел `გრამატიკა` много раз как «заглушку» и теперь узнаёт: «о, это слово значит "грамматика"!».

**Дофамин:** разблокировка = вознаграждение + обучение одновременно. Анимация появления нового раздела = ощущение «открыть дверь».

## Логика разблокировки

### Порядок секций (для beginners)

| Секция | Ключ | Georgian label | Разблокирована когда |
|--------|------|----------------|----------------------|
| Основы | `basics` | `საფუძვლები` | Всегда |
| Грамматика | `grammar` | `გრამატიკა` | Все модули секции `basics` пройдены на 100% |
| Лексика | `vocab` | `ლექსიკა` | Все модули секции `grammar` пройдены на 100% |
| Продвинутое | `advanced` | `გაღრმავება` | Все модули секции `vocab` пройдены на 100% |
| Мой словарь | `myvocab` | `ლექსიკონი` | Всегда (без условий) |

### Вычисление "раздел пройден"

Секция считается завершённой, когда для **каждого модуля** в секции:
`completedLessons[moduleId].length === module.lessons.length && module.lessons.length > 0`

Если у модуля нет уроков (my-vocabulary) — он не учитывается в подсчёте.

### Для intermediate

Все секции открыты с первого входа. Логика разблокировки не применяется.

### Сохранение состояния

Разблокированные секции сохраняются в `localStorage` ключ `bombora_unlocked_sections` как `string[]`. Раз разблокированная секция не блокируется обратно (даже если прогресс стёрт).

## Экраны и состояния

### Экран: Dashboard — Раздел заблокирован (Locked Section Tile)

**Layout:**
- Позиция: там же, где раздел был бы, если бы был открыт
- Высота: фиксированная 72px — «намёк» на содержимое
- Ширина: полная (с `px-5` паддингом как другие секции)

**Визуальный дизайн:**
```
┌─────────────────────────────────────────────────────┐  ← jewel-tile, opacity: 0.45
│  🔒  გრამატიკა          грамматика            N ур. │
│      ─────────── пройди «Основы», чтобы открыть ─── │
└─────────────────────────────────────────────────────┘
```

Детали:
- `jewel-tile` с `opacity-45` (глушим через CSS: `opacity: 0.45 + grayscale(30%)`)
- Иконка замка: SVG `🔒` 18×18px, jewelInk-hint цвет
- Грузинское название секции: `font-geo text-[18px] font-bold text-jewelInk-mid` — крупно, это обучающий элемент
- Русское название маленьким текстом рядом: `font-sans text-[13px] text-jewelInk-hint`
- Счётчик уроков `N ур.` — смазанный (jewelInk/20) — пользователь видит «там что-то есть»
- Подпись: `font-sans text-[11px] text-jewelInk-hint` — «пройди «Основы», чтобы открыть»
- Кнопкой **не является** — не кликабелен (`pointer-events: none`)
- Нет `jewel-pressable`, нет hover-эффектов

**Состояния locked tile:**
- Default: описано выше
- (нет других состояний — не интерактивен)

---

### Экран: Dashboard — Момент разблокировки (Unlock Toast)

Срабатывает автоматически: первый рендер после того как секция стала разблокированной (вычисляется при mount, сравнивается с `bombora_unlocked_sections` в localStorage).

**Поведение:**
1. Locked tile мгновенно заменяется открытым разделом (с модулями)
2. Поверх экрана (fixed overlay, `z-50`) на 2.2 секунды показывается unlock toast
3. Toast автоматически исчезает

**Unlock Toast layout:**
```
╔═══════════════════════════════════════╗
║                                       ║
║   [Kilim strip 8px]                   ║
║                                       ║
║     გრამატიკა                         ║  ← крупно, font-geo, 36px, jewelInk
║     грамматика                        ║  ← мелко, font-sans, 14px, jewel-mid
║                                       ║
║     [ раздел открыт ]                 ║  ← stamp-компонент с ruby
║                                       ║
║   [Kilim strip 8px]                   ║
║                                       ║
╚═══════════════════════════════════════╝
```

- Позиция: fixed bottom, full width, выше нижнего safe area
- Фон: `bg-cream border-t-[2px] border-jewelInk`
- Georgian название: `font-geo text-[36px] font-extrabold text-jewelInk mn-reveal` (анимация mn-reveal)
- Stamp: существующий компонент `Stamp`, цвет ruby, текст «раздел открыт»
- Появление toast: slide-up (translateY от 100% до 0, 300ms ease-out)
- Исчезновение: fade-out (opacity 1→0, 400ms, после 1.8s задержки)

**Анимации:**
- Toast появляется: `translateY(100%) → translateY(0)` за 300ms
- Georgian label внутри: `mn-reveal` (уже в CSS: scale+rotate анимация)
- Toast исчезает: автоматически через 2200ms, `opacity: 0` за 400ms

## Компоненты (новые)

### `LockedSectionTile`

**Назначение:** Показывает заблокированный раздел как визуальный намёк.

**Props:**
```typescript
interface LockedSectionTileProps {
  label: string        // 'грамматика'
  geoLabel: string     // 'გრამატიკა'
  lessonCount: number  // сколько уроков внутри
  prerequisite: string // 'пройди «Основы»'
}
```

**Варианты (variants):** только один вид — locked

**Привязка к Minankari-токенам:**
- Фон: `#FDFAEF` (jewel-tile base) с `opacity: 0.45`
- Граница: `border-jewelInk` с той же прозрачностью
- Тени: без shadow (или `box-shadow: none`) — нет «высоты»
- Иконка замка: `text-jewelInk-hint`

---

### `UnlockToast`

**Назначение:** Временное уведомление о разблокировке нового раздела.

**Props:**
```typescript
interface UnlockToastProps {
  geoLabel: string   // 'გრამატიკა'
  label: string      // 'грамматика'
  onDone: () => void // вызывается после исчезновения
}
```

**Поведение:**
- Монтируется → играет анимацию появления
- Через 1800ms → начинает fade-out
- Через 2200ms → вызывает `onDone()` → размонтируется

**Привязка к Minankari-токенам:**
- Фон: `cream #FBF6EC`
- Верхняя граница: `2px solid #15100A`
- Georgian label: `text-jewelInk`, `font-geo`
- Stamp: ruby accent

## Адаптивность

- Locked tile: `min-height: 72px` → достаточно для tap target (но tile не кликабелен)
- Toast: full width, `max-width` не нужен (он full-screen bottom strip)
- Georgian label в toast: 36px может быть много на 375px — клиппинг не страшен, это крупный акцент
- На 375px: все тексты помещаются, паддинги `px-5` стандартные

## Поведение для возвращающегося пользователя

**Сценарий:** пользователь прошёл Основы → вышел из аппа → зашёл снова.

1. При mount вычисляем `isUnlocked(section)` по текущему прогрессу
2. Сравниваем с `bombora_unlocked_sections` (localStorage)
3. Если секция стала разблокированной и её нет в localStorage → показываем toast + сохраняем в localStorage
4. Если уже в localStorage → тихо показываем открытый раздел без toast (не надоедаем повторно)

**Сценарий:** пользователь = intermediate.

Всё открыто, логика `LockedSectionTile` и `UnlockToast` не рендерится.

## Примечания для Developer

1. `isUnlocked(sectionKey, catalog, progress)` — чистая функция, логика вычисления по `completedLessons`
2. `bombora_unlocked_sections: string[]` в localStorage — массив ключей разблокированных секций
3. `UnlockToast` монтируется через портал (или просто fixed-position div в Dashboard)
4. Locked tile **не** рендерит секцию-заголовок с кнопкой collapse — вместо этого рендерит `LockedSectionTile`
5. Удалить существующую логику кнопки «показать все →» для beginners — она заменяется новым поведением
6. Анимация `mn-reveal` уже определена в `index.css` — использовать её для Georgian label в toast
