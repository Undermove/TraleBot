## Обучающий элемент

**Что учит:** Грузинский синтаксис — SOV-порядок слов (Субъект–Объект–Глагол) и правила постпозиций. Пользователь буквально строит грузинское предложение руками — это педагогически мощнее, чем выбрать ответ из списка.

**Три обучающих момента, встроенных в жест:**

1. **Порядок слов (SOV).** Слоты выстроены слева направо в грузинском порядке. Когда пользователь снова и снова кладёт глагол в правый крайний слот, он телесно усваивает SOV ещё до того, как прочитает про это в теории.
2. **Постпозиции.** Слот для постпозиции стоит ПОСЛЕ слота существительного (напр., «სახლ → _ში_»). Обратное русским предлогам. Жест «сначала слово, потом послелог» — это и есть урок.
3. **Reveal-момент.** На уроке «Постпозиции» пользователь вдруг узнаёт: «Вот почему я всё время ставил маленький кусочек справа от существительного!»

**Метки-обучалки над зонами (eyebrow T6, jewelInk-60):**
- Над пулом чипов: «სიტყვები» (слова) — пользователь видит это слово при каждом упражнении
- Над цепочкой слотов: «წინადადება» (предложение)

Reveal этих слов происходит в уроке «Алфавит» или «Лексика» — пассивное погружение до сознательного изучения.

**Рекомендуемая буква reveal для модуля Postpositions:** **შ** «шини» — маркер постпозиции «-ში» (в/на). Добавить в таблицу `LESSON_REVEAL_MAP` в `design-specs/68-letter-reveal-system.md`.

### Состояния экрана

**Default (нет выбранного чипа, часть или все слоты пустые):**
- Слоты: пустые → dashed jewelInk border
- Preset-слоты (L1–L3): navy fill, cream text — «эта позиция уже занята»
- Кнопка «Проверить»: disabled (`opacity: 40%`, cream фон, jewelInk border)
- Кнопка активируется только когда все незаполненные слоты заполнены

**Chip selected (пользователь тапнул чип в пуле):**
- Выбранный чип: gold fill, jewelInk border, scale 1.04 (80ms ease-out)
- Все пустые слоты: gold-pulsing outline, 600ms ease, 2 цикла
- Тап на пустой слот → чип перемещается туда
- Тап на заполненный слот → чипы меняются местами
- Тап на другой чип в пуле → переключает выделение

**Chip in slot (слот заполнен, нет выбранного чипа):**
- Чип в слоте: navy fill, cream text, 1.5px solid jewelInk border
- Тап на заполненный слот → чип возвращается в пул (scale 0.9→1.0, 100ms)

**After check — всё правильно:**
- Все слоты: navy bg, cream text, 2px solid jewelInk border
- FeedbackBanner (существующий компонент) «სწორია!» slide-up 200ms
- Кнопка меняется на «Далее» (navy fill, cream text)

**After check — есть ошибки:**
- Правильные слоты: navy fill, cream text
- Неправильные/пустые слоты: ruby fill, cream text, 2px solid jewelInk border
- Shake-анимация на неправильных слотах: 200ms
- Под каждым неправильным слотом: hint T6 (11px), ruby, однострочный
- Чипы в пуле (неиспользованные): opacity 40%, disabled
- Кнопка: «Далее» (navy fill) — можно продолжить даже с ошибками

**Error (данные не загрузились):**
- jewel-tile cream с текстом «Упражнение не загрузилось»
- jewel-btn «Попробовать снова»

### Компоненты (новые)

#### WordChip

Маленький вариант jewel-tile для грузинских слов в пуле.

```tsx
interface WordChipProps {
  text: string
  state: 'default' | 'selected' | 'in-slot' | 'correct' | 'incorrect' | 'disabled'
  onTap: () => void
}
```

| Состояние | Фон | Бордер | Тень | Текст |
|-----------|-----|--------|------|-------|
| `default` | cream | 1.5px jewelInk solid | 3px ink offset | jewelInk |
| `selected` | gold `#F5B820` | 1.5px jewelInk solid | 3px ink offset | jewelInk |
| `in-slot` | navy `#1B5FB0` | 1.5px jewelInk solid | нет | cream |
| `correct` | navy | 2px jewelInk solid | нет | cream |
| `incorrect` | ruby `#E01A3C` | 2px jewelInk solid | нет | cream |
| `disabled` | cream opacity 40% | 1.5px jewelInk opacity 30% | нет | jewelInk opacity 40% |

- Размер: min-height 44px, min-width 44px; по содержимому (но не меньше tap-target)
- Font: Manrope 600, T4 (15px) для обычных слов; `font-geo` (Noto Sans Georgian) для грузинского
- Padding: 10px 14px
- Border-radius: 8px

#### SentenceSlot

Слот-приёмник для чипа.

```tsx
interface SentenceSlotProps {
  state: 'empty' | 'active' | 'filled' | 'correct' | 'incorrect' | 'preset'
  chip?: { text: string }
  hint?: string          // однострочная подсказка под incorrect
  onTap: () => void
}
```

| Состояние | Фон | Бордер | Анимация |
|-----------|-----|--------|----------|
| `empty` | cream | 1.5px dashed jewelInk | нет |
| `active` | cream | 2px solid gold, box-shadow 0 0 0 3px gold-30% | pulse 600ms |
| `filled` | прозрачный | — | нет (chip внутри) |
| `correct` | — | 2px solid navy | нет |
| `incorrect` | — | 2px solid ruby | shake 200ms |
| `preset` | navy | 1.5px solid jewelInk | нет |

- Размер: height 48px, min-width 52px; растягивается по длине вложенного чипа
- Border-radius: 8px
- Gap между слотами: 8px

Hint под `incorrect`:
```
слот [неправильное слово]
     └─ «нужен дательный падеж»  ← T6 (11px), ruby, Manrope 500
```

#### SentenceSlotRow

Горизонтальная цепочка SentenceSlot. Overflow-x: auto со snap (если слотов больше 5).

#### ChipPool

Wrapping flex-контейнер для WordChip. Все чипы видны без скролла (max ~15 чипов).

### Интеграция в Practice.tsx

По аналогии с `AudioChoiceCard`:

```tsx
// В Practice.tsx
const isSentenceBuilder = current.questionType === 'sentence-builder'

// В JSX question-area:
{isSentenceBuilder ? (
  <SentenceBuilderCard
    key={current.id}
    question={current}
    onAnswer={(isCorrect) => handleAnswer(isCorrect)}
  />
) : (
  /* existing question render */
)}
```

`SentenceBuilderCard` управляет своим состоянием chip selection / slot filling самостоятельно. Вызывает `onAnswer(isCorrect)` при нажатии «Далее».

## JSON-схема вопроса

```json
{
  "questionType": "sentence-builder",
  "targetSentence": { "ru": "Я иду в дом" },
  "level": 1,
  "correctOrder": ["მე", "სახლში", "მივდივარ"],
  "chipPool": ["მე", "სახლში", "მივდივარ", "ვარ", "სახლი", "გახვედი", "-ში", "მოდი"],
  "presetPositions": [
    { "position": 0, "token": "მე" },
    { "position": 2, "token": "მივდივარ" }
  ],
  "hints": {
    "1": "Постпозиция -ში обозначает «в» и стоит после существительного"
  }
}
```

- `correctOrder` — массив правильных токенов в правильном порядке (defines slot count)
- `chipPool` — все чипы для пула, включая дистракторы
- `presetPositions` — предзаполненные позиции (для L1–L3; пустой массив для L5+)
- `hints` — словарь `{ "индекс_слота": "текст подсказки" }` (только для типичных ошибок)

## Прогрессия уровней

| Уровень | Пресет слотов | Кол-во чипов в пуле | Сложность |
|---------|---------------|---------------------|-----------|
| L1 | Все, кроме 1 | 4 | 1 пустой слот, найди правильную форму |
| L2 | Все, кроме 2 | 6–8 | 2 слота, проверяет согласование |
| L3 | Подлежащее + сказуемое | 8–10 | Дополнения и обстоятельства |
| L4 | Только глагол | 10–12 | Остальное собрать вокруг глагола |
| L5+ | Нет пресетов | 12–15 | С нуля, 8–10 правильных + дистракторы |

## Адаптивность (375px)

- Эталонная плитка: padding 16px, текст T2 (22px) — помещается в 2 строки если длинная фраза
- SentenceSlotRow: overflow-x: auto + scroll-snap при >5 слотах; visible indicator если есть скролл
- ChipPool: flex-wrap, 8px gap — при 15 чипах занимает 3–4 строки, что нормально
- WordChip min-height и min-width 44px — tap target соблюдён
- SentenceSlot height 48px — tap target соблюдён
- jewel-btn «Проверить»/«Далее»: full-width, height 52px
