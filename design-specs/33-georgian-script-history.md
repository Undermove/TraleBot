# История грузинского письма — культурная карточка в Алфавите

**Задача:** ROADMAP.md → задача #33
**Статус:** ready

---

## Обучающий элемент

**Паттерн:** пользователь учит буквы, не зная, что держит в руках один из старейших алфавитов мира. Культурные факты — поданные в нужный момент — превращают механическое запоминание знаков в эмоциональное открытие.

**Вопросы дизайна:**

1. **Что тут можно показать на грузинском?** Три названия стилей письма: **მხედრული** (мхедрули, «воинское»), **ასომთავრული** (асомтаврули, «заглавное»), **ნუსხური** (нусхури, «строчное»). Само слово **ისტორია** (история). Число **V** — пятый век — записывается через грузинские числительные символы.

2. **Когда наступит reveal-момент?**
   - В карточке 4: пользователь узнаёт, что буква **ქ** в лоадере — первая буква слова **ქართული** («грузинский»). Он видел её при каждом запуске приложения — и только сейчас понял, что читал это слово всё время.
   - В уроке про букву **ქ** (RevealKaniOverlay): уже известна backstory → двойной дофамин.

3. **Как это создаёт дофамин?** Удивление «алфавиту 1600 лет»; гордость «я учу один из 14 уникальных алфавитов мира»; узнавание лоадера → «я это видел с первого дня».

- При свайпе влево — обратное направление

### Компонент: HistoryCarousel (новый)

**Назначение:** полноэкранный оверлей-карусель. Управляет слайдами и жестами.

**Props:**
```typescript
interface HistoryCarouselProps {
  onClose: () => void
}
```

**Внутреннее состояние:**
```typescript
const [currentSlide, setCurrentSlide] = useState(0)        // 0-3
const [direction, setDirection] = useState<'next' | 'prev'>('next')
const [animating, setAnimating] = useState(false)
```

**Логика навигации:**
```typescript
const goTo = (idx: number, dir: 'next' | 'prev') => {
  if (animating) return
  setDirection(dir)
  setAnimating(true)
  setTimeout(() => {
    setCurrentSlide(idx)
    setAnimating(false)
  }, 240)
}

const next = () => currentSlide < 3 && goTo(currentSlide + 1, 'next')
const prev = () => currentSlide > 0 && goTo(currentSlide - 1, 'prev')
```

**Swipe-обработчик:**
```typescript
const touchStart = useRef(0)
const onTouchStart = (e: React.TouchEvent) => {
  touchStart.current = e.touches[0].clientX
}
const onTouchEnd = (e: React.TouchEvent) => {
  const delta = touchStart.current - e.changedTouches[0].clientX
  if (Math.abs(delta) > 40) {
    delta > 0 ? next() : prev()
  }
}
```

**Привязка к Minankari-токенам:**
- Фон оверлея: `bg-cream`
- Карточка: `.jewel-tile`
- Кнопки: `.jewel-btn .jewel-btn-cream` (← Назад) и `.jewel-btn .jewel-btn-navy` (Далее →)
- Закрыть (×): T4, `text-jewelInk-mid`, tap target `44×44px`
- Точки: `w-2 h-2 rounded-full`, active = `bg-gold`, inactive = `bg-jewelInk/20`

## Адаптивность (375px)

- Оверлей: `position: fixed inset-0` — всегда полный экран
- Карточка: `w-[90%] mx-auto`, `max-h-[calc(100dvh-200px)]`, `overflow-y: auto` на случай очень длинного контента
- Три мини-плитки на карточке 2: `flex gap-2`, каждая `flex-1` → ~110px на 375px ✓
- Кнопки навигации: flex row, full-width с `gap-3`, обе `flex-1 min-h-[52px]` ≥ 44px ✓
- Буква ქ: `display-xl` = `clamp(44px, 12vw, 64px)` → 44px на 375px ✓
- Кнопка закрыть ×: `w-[44px] h-[44px]` ≥ 44px ✓
- Штамп на карточке 4: `max-w-[200px] mx-auto` — не выходит за края ✓
