# История грузинского письма — культурная карточка в Алфавите

**Задача:** ROADMAP.md → задача #33
**Статус:** ready

---

## Обучающий элемент

Пользователь учит алфавит, но не знает, что он прикасается к одному из старейших письменных систем мира. Исторический контекст превращает «набор из 33 знаков» в живую культурную традицию — мотивация резко возрастает.

**Что показывается на грузинском:**
- Названия трёх стилей письма: **მხედრული**, **ასომთავრული**, **ნუსხური**
- «V საუკუნე» — V век на грузинском (готовит к уроку числительных)
- Буква **ქ** как главный reveal — первая буква слова **ქართული** («грузинский»)

**Когда наступит reveal-момент:**
Двойной reveal-момент:
1. Пользователь видел ქ в лоадере каждый день → история рассказывает, что это первая буква ქართული → эмоция «теперь я понимаю, что видел!»
2. Если пользователь уже видел `RevealKaniOverlay` в уроке Алфавита — карточка истории это закрепляет и расширяет.

**Как создаётся дофамин:**
Четыре последовательных открытия: «это так старо», «тут три разных стиля!», «без заглавных вообще?», «та самая ქ!» — каждая карточка = отдельный момент удивления.

## Экраны и состояния

### Точка входа: кнопка-баннер «История алфавита»

**Где:** в `ModuleMap.tsx`, только для alphabet-модулей (`moduleId === 'alphabet-progressive' || moduleId === 'alphabet'`).

**Позиция:** между `ModulePhraseBanner` и секцией «уроки». Под `<div className="mn-eyebrow mb-3">уроки</div>` не добавлять — баннер идёт ДО этого eyebrow.

**Layout (375px):**

```
┌────────────────────────────────────────────┐
│ [═══ Kilim overview card ════════════════] │
│                                            │
│ [ModulePhraseBanner — если ещё не видел]   │
│                                            │
│ ╔══════════════════════════════════════╗   │  ← HistoryBanner (jewel-tile)
│ ║ [Бомбора 36px cheer]                ║   │
│ ║ История грузинского письма     [→]  ║   │    стрелка navy
│ ║ V საუკუნე · 4 карточки             ║   │    eyebrow-строка, gold-deep
│ ╚══════════════════════════════════════╝   │
│                                            │
│  ── уроки ──                               │
│  ...карта уроков...                        │
└────────────────────────────────────────────┘
```

**Детали баннера:**
- `jewel-tile jewel-pressable` + `px-4 py-3`
- Flex: Бомбора слева (36px, mood=cheer), текстовый блок справа, стрелка `→` крайняя правая
- Заголовок: `font-sans text-[15px] font-extrabold text-jewelInk` (T4)
- Подпись «V საუკუნე · 4 карточки»: `mn-eyebrow` стиль (11px bold uppercase) цвет `text-gold-deep`
- Стрелка `→`: `text-navy font-bold text-[16px]`
- `min-h-[60px]` — tap target ≥ 44px ✓

**Состояния баннера:**
- **Default** — показывается всегда для alphabet-модулей (не скрывается после просмотра — это ценный ресурс, не one-shot)
- **Pressed** — jewel-pressable: translate(3px, 3px) + shadow → 0, 80ms

#### Оверлей и фон

- Фон оверлея: `bg-cream` (не тёмный overlay — это светлый экран с информацией, не блокирующий модальник)
- Килим-страйп сверху и снизу — стандартный, как на всех экранах
- Анимация появления: `page-in` (translateY(18px) → translateY(0), 320ms ease-out, уже в index.css)
- Анимация закрытия: обратная — translateY(0) → translateY(18px), opacity 1→0, 250ms ease-in

#### Шапка

```
[ ×  закрыть ]    [ текст не нужен ]    [ 1 / 4 ]
```

- `×`: `w-9 h-9 flex items-center justify-center rounded-full border border-jewelInk/20 text-[18px] text-jewelInk active:bg-jewelInk/10`
- Счётчик `1 / 4`: `font-sans text-[12px] font-bold text-jewelInk-mid` (T6)
- Строка шапки: `flex items-center justify-between px-5 pt-4 pb-2`

#### Карточка контента (центральная зона)

- Тип: `jewel-tile` (кремовый фон, ink border, gold hairline внутри)
- Размер: `mx-5 flex-1` — занимает всё доступное пространство между шапкой и навигацией
- `px-5 py-5`

**Структура карточки (типовая для карточек 1, 3):**

```

**Структура карточки 2 (три стиля):**

```
[ Бомбора 56px, mood=think ]

[ სამი სტილი — три стиля   T3, 18px, navy ]
[ divider ]

Три грузинских буквы рядом (например ა):

┌──────────┬────────────────┬──────────┐
│ [ა mxed] │ [Ⴀ asomtav]   │ [ა nusk] │
│ T1, 36px │ T1, 36px       │ T1, 36px │
│ navy     │ ruby           │ gold-deep│
│ მხედ-    │ ასომ-          │ ნუსხ-    │
│ რული     │ თავრული        │ ური      │

Примечание: Noto Sans Georgian поддерживает все три стиля. Шрифт для ასომთავრული и ნუსხური через CSS `unicode-range` или просто использование символов соответствующего Unicode-диапазона (Asomtavruli: U+10D0 → U+10FF, Nuskhuri: U+2D00–U+2D2F, Mkhedruli: U+10D0–U+10FF обычные). Для простоты реализации — показывать все три как разные символы без кастомного `font-face`: ა (мхедрули, обычный) / Ⴀ (ასომთავრული) / ⴀ (ნუსხური). Developer должен проверить рендеринг в Noto Sans Georgian, при необходимости использовать SVG-иллюстрацию.

**Структура карточки 4 (reveal ქ):**

[ divider ]

Буква ქ с золотым подчёркиванием: `font-geo text-[64px] font-bold text-navy` с `border-b-[3px] border-gold` или инлайн-style `borderBottom: '3px solid #F5B820'`.

#### Навигация (нижняя зона)

```
║  [● ○ ○ ○]     [Дальше →]        ║
```

- Точки прогресса: `flex gap-2`
  - Активная: `w-3 h-3 rounded-full bg-navy`
  - Неактивная: `w-2 h-2 rounded-full bg-jewelInk/20`
- Кнопка «Дальше →» / «Запомню! ✓»:
  - `jewel-btn jewel-btn-navy` для «Дальше»
  - `jewel-btn jewel-btn-ruby` для «Запомню!» (финальная — ruby как цвет завершения)
  - Ширина: фиксированная `min-w-[140px]`

Кнопка «Назад» — отсутствует намеренно. Свайп-навигация опциональна (touch event на карточку). Главный путь — линейный, как страницы книги.

#### Переход между карточками

- Анимация: текущая карточка fade+slide влево (translateX(0) → translateX(-20px), opacity 1→0, 200ms),
  следующая появляется справа (translateX(20px) → translateX(0), opacity 0→1, 250ms, delay 150ms)
- Класс: `.mn-card-exit` и `.mn-card-enter` (добавить в index.css)

#### Состояния компонента ScriptHistoryCarousel

| Состояние | Описание |
|-----------|----------|
| **hidden** | Компонент не рендерится, пока баннер не нажат |
| **open:card-N** | Показана карточка N (1-4), навигация активна |
| **closing** | Анимация закрытия (translateY), 250ms |
| **closed** | Компонент размонтирован, возврат к ModuleMap |

localStorage-флаг не используется — карусель открывается каждый раз при нажатии баннера.

## Адаптивность (375px)

- Баннер: ширина `w-full`, Бомбора 36px + текстовый блок + стрелка → суммарно ~ 345px без overflow ✓
- Карточки: `mx-5` = 335px рабочей ширины, `px-5 py-5` = 295px для текста
- Карточка 2 (три стиля): 3 колонки по ~90px = 270px, суммарно помещается в 295px ✓
- Буква ქ 64px: высота ~ 70px + золотая линия = комфортно ✓
- Навигационная строка: dots (4×12px + gaps = ~60px) + кнопка (140px min) — суммарно ~220px на строке 335px ✓
- Тело карточки с текстом: максимум ~6 строк при 14px = ~104px. Все 4 карточки умещаются без скролла на экране 667px (iPhone SE) ✓
- Tap targets: `×` 36px + p-1.5 расширение = ~42px → достаточно; кнопка «Дальше» 56px min-height ✓
