# Аорист: расширение — 1л. и 2л. («я написал», «ты написал»)

**Задача:** ROADMAP.md → задача #41 · GitHub issue #313
**Статус:** ready

---

## Контекст проблемы

Текущий модуль «Аорист» (5 уроков) учит только формы **3-го лица**:
- L1: «ის წავიდა» (он ушёл) — Class 2, без переворота
- L2: «კაცმა დაწერა» (мужчина написал) — Class 1, переворот эргатива
- L3–L5: частые глаголы, неправильные, повторение — везде 3л.ед.

После прохождения модуля ученик **не может сказать «я написал»** (დავწერე) или **«ты написал»** (დაწერე). Это критический разговорный пробел: 1л. и 2л. аориста — самые частые форм в рассказах о прошлом («Что ты делал вчера?», «Я сходил в...», «Ты видел?»).

---

## Обучающий элемент

### Reveal-момент: -ვ- снова!

Ученик уже знает **-ვ- как маркер 1-го лица** из настоящего времени:
- ვწერ — «я пишу» (-ვ- в начале)
- წერს — «он пишет» (без -ვ-)

В аористе этот **же самый -ვ-** присутствует:
- **დავ**წერე — «я написал»
- **დვ**ამბობდი... — аналогично в других временах

Reveal: «-ვ- это всегда я!» Система глагольных маркеров лиц **единообразна** — то, что работало в настоящем, работает и в аористе. Это снимает ощущение хаоса и даёт ученику инструмент для самостоятельного анализа незнакомых форм.

### Вторичный момент: -ა vs -ე на конце

Ученик видел 3л. аориста: დაწერ**ა** (он написал). В 1л./2л. окончание меняется: დავწერ**ე** (я написал), დაწერ**ე** (ты написал). Одна буква — другое лицо. Это закрепляет понимание личных окончаний как системы.

## Содержание теории — урок 6

**Lesson metadata:**
```
number: 6
title: «Я написал» — 1л. и 2л. аориста
subtitle: "დავწერე / დაწერე"
goal: "Говорить о своих прошлых действиях"
description: "Знакомый -ვ- появляется в аористе тоже!"
```

**Theory blocks (порядок):**

**Block 1 — Paragraph (reveal):**
> Ты уже знаешь -ვ- из настоящего времени: **ვ**წერ = я пишу. В аористе — та же логика. **და**-**ვ**-წერ-**ე** = я написал. Один маркер, одно значение: -ვ- всегда = «я».

**Block 2 — List (парадигма 1л.ед./2л.ед./3л.ед.):**
```
Аорист «написал» (переходный, да-წერ-):
• 1л.ед.: (მე) და-ვ-წერ-ე — я написал          ← -ვ- внутри
• 2л.ед.: (შენ) და-წერ-ე — ты написал           ← без -ვ-, но то же окончание -ე
• 3л.ед.: (მან) და-წერ-ა — он/она написал(а)    ← -ა на конце (уже знакомо!)
```

**Block 3 — Paragraph (паттерн окончаний):**
> Три лица — три окончания: **-ე** для 1л. и 2л., **-ა** для 3л. В настоящем -ვ- стоит снаружи (ვ-წერ). В аористе -ვ- внутри между превербом и корнем: და-**ვ**-წერ-ე.

**Block 4 — List (множественное число):**
```
Множественное число:
• 1л.мн.: (ჩვენ) და-ვ-წერ-ეთ — мы написали    ← -ვ- + окончание -ეთ
• 2л.мн.: (თქვენ) და-წერ-ეთ — вы написали      ← -ეთ без -ვ-
• 3л.мн.: (მათ) და-წერ-ეს — они написали        ← -ეს на конце
```

**Block 5 — Example (частотный диалог):**
```
Georgian: — გუშინ რა გააკეთე?
Translit: — gushin ra gaakete?
Russian:  — Что ты сделал вчера?
```

**Block 6 — Example (ответ с 1л.):**
```
Georgian: — ვიარე ბაზარში და ვიყიდე პური
Translit: — viari bazarshi da viqide puri
Russian:  — Я сходил на рынок и купил хлеб
```

**Block 7 — Paragraph (правило эргатива для 1л./2л.):**
> Переворот эргатива работает и в 1л./2л.! «Я написал письмо»: **მე** остаётся в именительном — потому что у 1л./2л. именительный = эргативный (синкретизм, как в PresentTense L2). Только 3л. меняет форму: ის → მან.

## Методистская проверка

**Требования до мёржа:**

1. **Эргативный синкретизм 1л./2л.** — в теории обязательно объяснить, что у «მე/შენ» форма именительного = эргативного (Block 7). Без этого ученик запутается: «почему нет -მ-?»

2. **Не перегружать** — урок покрывает только Class 1 (переходные, с превербом). Class 2 в аористе без переворота и так очевиднее (ის წავიდა → მე წავედი). Если добавлять — отдельным параграфом, не основной темой.

3. **Частотные глаголы** — примеры только из глаголов, знакомых из L3 («Частые аористы»): дав-წერ-е, дав-ი-ყიდ-ე, გავ-аკეთ-е, дав-ли-е, წав-ი-კითხ-е. Не вводить новые глаголы.

4. **Однозначность форм** — 2л.ед. «დაწერე» и 2л.мн. «დაწერეთ» различаются только окончанием (-ე vs -ეთ). Это потенциальная точка спутывания. В теории и квизах явно маркировать «ед.ч.» / «мн.ч.».

## Адаптивность (375px)

Параграф с парадигмой (Block 2 и 4) содержит строки вида:
```
• 1л.ед.: (მე) და-ვ-წერ-ე — я написал
```
На 375px это ~55–60 символов. При стандартном `font-size: 14px` и `line-height: 1.5` строка может не вместиться в одну строку. Рекомендация:

- Использовать `List`-блок (уже есть в existing TheoryBlockDto) — он рендерит каждый элемент как `<li>`, обёрточно.
- Если грузинский текст с транслитерацией слишком длинный — разбить на два List-пункта: «1л.ед.: **დავწერე**» и «— я написал (да- + -ვ- + корень + -ე)».

Все tap targets в quiz (кнопки вариантов) — `min-h-[48px]`, уже соответствуют дизайн-системе.
