# VoM questions10.json tips: заменить бенгальские символы на грузинские

**Задача:** ROADMAP.md → задача #50 · GitHub issue #329  
**Статус:** ready

---

## Контекст проблемы

В `GeorgianVerbsOfMovement/questions10.json`, поле `intro.tips[1]`, два символа преверба `მო-` написаны бенгальскими символами Unicode вместо грузинских:

**Текущий (ошибочный) текст `tips[1]`:**
```
"Запомните приставки для направлений: всё зависит от মо-/მი-/შე-/გა-/ა-/ჩა-/გად-"
```

- `মো-` — первый преверб: символы U+09AE (ম, бенгальская «ма») + U+09CB (ো, бенгальская «о») + U+002D.
- Визуально при многих шрифтах выглядит идентично грузинскому `მო-`, но является другим Unicode-диапазоном (Bengali block U+0980–U+09FF вместо Mkhedruli U+10D0–U+10FF).

**Последствия:**
- Поиск и тесты могут давать несовпадение: грузинский `მ` (U+10DB) ≠ бенгальский `ম` (U+09AE).
- Ученик, скопировавший преверб из подсказки, получит бенгальские символы вместо грузинских.
- Итоговый урок L10 (единственная сводная подсказка по превербам) содержит фактическую ошибку в данных.

---

## Обучающий элемент

Не применимо — это исправление data-quality без новой обучающей механики. Корректные грузинские символы сами по себе поддерживают учёбу: ученик может безопасно копировать, искать и сверять превербы с теорией.

## Что меняется

### Файл: `src/Trale/Lessons/GeorgianVerbsOfMovement/questions10.json`

**Поле:** `intro.tips[1]`

**Было:**
```json
"Запомните приставки для направлений: всё зависит от মо-/მი-/შე-/გა-/ა-/ჩა-/გად-"
```
*(первые два символа после «от » — бенгальские U+09AE + U+09CB)*

**Стало:**
```json
"Запомните приставки для направлений: всё зависит от მო-/მი-/შე-/გა-/ა-/ჩა-/გად-"
```
*(все символы — чистые Mkhedruli: მ U+10DB + ო U+10DD)*

**Требования к правке:**
1. Заменить ровно два символа: U+09AE → U+10DB (`მ`), U+09CB → U+10DD (`ო`).
2. Остальные превербы в той же строке (მი-, შე-, გა-, ა-, ჩა-, გად-) — не трогать (они уже корректные Georgian).
3. Больше никакие поля `questions10.json` не меняются.
