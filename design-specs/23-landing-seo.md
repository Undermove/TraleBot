## Обучающий элемент

**Паттерн: «Подглядываем до начала»**

На лендинге есть секция «Ты научишься говорить» — горизонтальная прокручиваемая лента из 6 карточек-слов. Каждая карточка показывает грузинское слово крупно + маленький hint-перевод под ним (не прятать — задача создать узнавание).

Слова: გამარჯობა (привет), მადლობა (спасибо), კარგი (хорошо), სახლი (дом), მე მქვია (меня зовут), ბოდიში (извини).

**Reveal-момент:** пользователь начинает курс «Выживание A0→A1» и встречает точно такие же слова — «о, я это видел на сайте». Двойное узнавание: буква на лоадере قق, слово на лендинге. Создаёт ощущение «я это уже знаю немного».

**Дополнительно:** числитель уроков в секции программы обучения использует грузинские буквы-числительные (ა, ბ, გ, დ, ე) — те же, что в мини-аппе на дашборде. Пользователь видит ссылку ещё до установки.

---

## Архитектурная заметка для Developer

Это SPA (React/Vite). Поисковики плохо индексируют контент, который рендерится только через JS. Решение: **использовать уже имеющиеся мета-теги** в `index.html` (они там уже есть и хорошие) + убедиться, что `<noscript>` fallback содержит ключевой текст. Полная SSR не нужна — достаточно грамотных мета-тегов и структурированных заголовков.

Контент лендинга должен быть в DOM сразу при монтировании (без skeleton/lazy) — не использовать `Suspense` обёртки для LandingScreen.

### Секция: Kilim — полная ширина

**Проблема:** `#root` ограничен `max-width: 480px`. Килим-полосы должны выходить за эту границу на весь вьюпорт.

**Решение:** новый класс `mn-kilim-full`:

```css
/* В index.css добавить: */
.mn-kilim-full {
  height: 8px;
  position: relative;
  left: 50%;
  right: 50%;
  margin-left: -50vw;
  margin-right: -50vw;
  width: 100vw;
  background-image: /* тот же паттерн, что у .mn-kilim */;
  background-repeat: repeat-x;
  background-size: 32px 8px;
}
```

Применяется только на LandingScreen (мини-апп внутри Telegram использует обычный `mn-kilim`).

### Секция 3: Как это работает

**H2:** `Как это работает`

Три шага в вертикальной стопке, соединённые тонкой пунктирной линией `ink-dash`:

```
  [ა] Запусти бота → «🐶 Бомбора» в меню
      mn-eyebrow ← шаг один
      Бот просит выбрать уровень: с нуля или уже знаю алфавит

  ···· ink-dash ····

  [ბ] Открой мини-апп → начинается первый урок
      mn-eyebrow ← шаг два
      Теория с карточками, затем квиз на узнавание

  ···· ink-dash ····

  [გ] Учи в своём темпе
      mn-eyebrow ← шаг три
      Все уроки открыты. Повторяй, пропускай, возвращайся
```

**Детали:**
- Номер шага: грузинская буква-числительная `ა/ბ/გ` в синем квадрате 40×40px (`bg-navy text-cream rounded-lg border-[1.5px] border-jewelInk` + 2px offset shadow)
- `mn-eyebrow` лейбл: `шаг один / шаг два / шаг три` — строчными, без uppercase
- Текст шага: `font-sans text-[15px] font-extrabold` (основная мысль) + `text-[13px] text-jewelInk-soft` (пояснение)
- `ink-dash` разделитель: 1px горизонтальная пунктирная линия, `opacity-30`, между шагами
- Layout: `flex items-start gap-4` по горизонтали (иконка + текст)

### Секция 5: Программа обучения

**H2:** `Программа A0 → B2`

Accordion-стиль: каждый уровень — сворачиваемый блок. По умолчанию открыт только первый (A0).

```
[ ▼ A0 — Алфавит ] ← открыт
  ა Гласные · 5 букв
  ბ Лёгкие согласные · 5 букв
  გ Частые согласные · 5 букв
  დ Тройки · 5 букв
  ე Оставшиеся · 11 букв

[ ▶ A0→A1 — Выживание ] ← свёрнут
[ ▶ A1 — Числительные ] ← свёрнут
[ ▶ A1 — Основы грамматики ] ← свёрнут
[ ▶ A1-A2 — Лексика по темам ] ← свёрнут
[ ▶ A2→B1 — Глаголы в системе ] ← свёрнут
[ ▶ B1→B2 — Продвинутая грамматика ] ← свёрнут
```

**Заголовок уровня:**
- `font-sans text-[15px] font-extrabold text-jewelInk`
- Level badge (A0, A1 и т.д.): `mn-eyebrow text-navy text-[9px] bg-navy/10 px-1.5 py-0.5 rounded-sm`
- Chevron: `▶/▼`, transition 200ms ease-out

**Строки уроков внутри (открытый уровень):**
- Префикс: грузинская буква-числительная (ა, ბ, გ...) `font-geo text-[13px] text-navy font-bold w-5`
- Название: `font-sans text-[13px] text-jewelInk`
- Количество: `font-sans text-[11px] text-jewelInk-soft`
- `flex items-center gap-3 py-2 border-b border-jewelInk/8 last:border-0`

**Контейнер accordion:** `jewel-tile divide-y divide-jewelInk/10` (единая плитка со внутренними разделителями)

**H2 суб-хинт:** под заголовком «Программа» — `font-sans text-[13px] text-jewelInk-soft` — «Все уроки открыты сразу — выбирай с любого места»

## Адаптивность (375px)

- `H1` font-size: `text-[32px]` на 375px (вместо `text-[36px]`) — через Tailwind responsive или clamp
- WordTeaser карточки: `w-[130px]` на 375px (слегка компактнее)
- FeatureTile (секция 6): без изменений, уже работает на 375px
- Accordion секции: нет изменений — вертикальный layout, всегда вписывается
- FAQ: без изменений
- CTA кнопка: `w-full max-w-[320px]`, на 375px занимает почти всю ширину
- Все tap targets ≥ 44px: accordion header `min-h-[44px]`, FAQ items `min-h-[44px]`, CTA кнопка `min-h-[56px]`
- kilim-full ширина: на любом экране 100vw — корректно
