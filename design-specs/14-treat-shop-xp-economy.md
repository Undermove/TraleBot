## Обучающий элемент

**Грузинские слова — названия еды:**

Каждая вкусняшка называется по-грузински. Пользователь видит эти слова каждый раз при открытии магазина:

| Emoji | Русское | Грузинское | Транслитерация |
|-------|---------|------------|---------------|
| 🦴 | Косточка | **ძვალი** | dz-va-li |
| 🥩 | Мяско | **ხორცი** | khor-ci |
| 🍖 | Мцвади | **მწვადი** | mts-va-di |
| 🎂 | Чурчхела | **ჩურჩხელა** | chur-chkhe-la |
| 👑 | Супра | **სუფრა** | su-pra |

Первые два — базовые слова из словарного запаса. Последние три — ключевые культурные реалии Грузии:
- **მწვადი** (шашлык по-грузински) встречается в модуле «Кафе» (Фаза 4)
- **ჩურჩხელა** (орехи в виноградном соке) — иконический грузинский продукт
- **სუფრა** (торжественный стол, грузинское застолье) — центральная культурная концепция

**Reveal-момент:** пользователь, купивший «Мцвади» или «Чурчхела» для Бомборы, встречает те же слова в модуле «Кафе» или будущем культурном блоке — «я уже знаю это слово, я кормил им Бомбору!». Двойной дофамин: заботливый ритуал → узнавание в учебном контексте.

**Паттерн:** игровой элемент (кормёжка) → грузинское название → reveal в лексическом модуле.

## Экономика XP

Доступные XP = `progress.xp - progress.xpSpent` (новое поле `xpSpent` в `ProgressState`, синхронизируется с бэкендом).

| Вкусняшка | Цена XP | Анимация |
|-----------|---------|---------|
| 🦴 ძვალი — Косточка | **10 XP** | Бомбора грызёт (мелкое жевание, mood stay happy) |
| 🥩 ხორცი — Мяско | **30 XP** | Бомбора прыгает от радости (3× bounce, mood=cheer) |
| 🍖 მწვადი — Мцвади | **60 XP** | Бомбора танцует + конфетти с боков |
| 🎂 ჩურჩხელა — Чурчхела | **100 XP** | Фейерверк + золотые искры + Бомбора в восторге |
| 👑 სუფრა — Супра | **200 XP** | Золотой дождь + Бомбора в короне + вся экономика анимации |

## Экраны и состояния

### Экран: Dashboard — кнопка входа в магазин

Добавляется компактная кнопка-строка **под hero-секцией Бомборы** (после suggestion-карточки или вместо неё, если suggestion нет).

**Layout:**

```
[ hero — Бомбора + приветствие ]
[ suggestion card (если есть) ]

────────────────────────────────────
[ 🦴 Покормить    ╱   N xp свободно ]  ← кнопка-строка
────────────────────────────────────
```

**Кнопка «Покормить»:**
- Компонент: `jewel-tile jewel-pressable`
- Высота: 52px (tap target ≥ 44px)
- Полная ширина (`w-full`)
- Два столбца: левый — иконка 🦴 (24px) + текст «Покормить Бомбору» (T4, font-bold); правый — XP-чип
- **XP-чип** (правый): `[N xp]` — `bg-gold/15 text-jewelInk border border-gold rounded-full px-2 py-0.5 text-[12px] font-extrabold`
- Горизонтальные отступы: `px-4`

**Состояния кнопки:**
- Default: описано выше
- `xpSpent >= xp` (0 доступных XP): иконка меняется на 💤, текст «Зарабатывай XP в уроках», чип `0 xp` серый; кнопка остаётся тапабельной (открывает магазин в режиме «пусто»)

**Детали bottom sheet:**
- Background: `bg-cream` (`#FBF6EC`)
- Border: `border-t-2 border-x-2 border-jewelInk rounded-t-2xl`
- Shadow: `0 -4px 0 #15100A` (верхний offset shadow — стандарт Minankari)
- Slide-up: `translateY(100%) → translateY(0)`, 300ms ease-out
- Backdrop: `rgba(21,16,10,0.4)`, fade 200ms; тап на backdrop закрывает шторку

**Заголовок-блок:**
- Бомбора (`Mascot`, 72px) — слева в ряду; текст-столбик справа
- «Покормить Бомбору» — T2 font-extrabold
- XP-чип под заголовком: `[⚡ N xp доступно]`
  - bg: `#F5B820/15`, border: `1px solid #F5B820`, text: `#15100A`
  - `text-[13px] font-extrabold px-3 py-1 rounded-full inline-flex items-center gap-1`

**Список вкусняшек:**

Каждый элемент — отдельная строка-плитка (`jewel-tile`), 60px высота:

```
┌────────────────────────────────────────┐
│  [emoji 28px]  [Название-geo  ]  [XP]  [+]  │
│              [ru-name T5 hint  ]       │
└────────────────────────────────────────┘
```

Структура строки:
- Emoji (28px)
- Текстовый столбик:
  - Первая строка: Georgian-название (T4, font-bold, `font-geo`) 
  - Вторая строка: русское название (T5, `text-jewelInk/50`, italic-free)
- XP-цена: `[N xp]` — `text-[13px] font-extrabold text-jewelInk`
- Кнопка «+» (`jewel-btn-gold`, 36×36px, rounded-full) — справа

**Состояния строки вкусняшки:**

| Состояние | Описание |
|-----------|----------|
| Доступна (xp >= price) | Полная яркость. Кнопка «+» активна (gold fill). |
| Недоступна (xp < price) | Opacity 0.45. Кнопка «+» disabled (серая). |
| Загрузка (после тапа) | Кнопка «+» заменяется на `LoaderLetter` 18px (ქ крутится). |
| Успех | Кнопка «+» → ✓ на 1.5 сек, затем XP-баланс пересчитывается. |

### Экран: FeedAnimation — оверлей анимации кормёжки

После успешного списания XP — полноэкранный (или 60% экрана) анимационный оверлей поверх Dashboard/TreatShop.

**Структура оверлея:**
```
┌──────────────────────────────────────────┐
│  [ backdrop полупрозрачный 60% ]         │
│                                          │
│        [ Бомбора 160px АНИМАЦИЯ ]        │
│                                          │
│  [ эмодзи угощения большой, 48px ]      │
│  [ Georgian название, T2 ]              │
│  [ транслитерация, T5, hint ]           │
│                                          │
│  [ ЧАСТИЦЫ / КОНФЕТТИ по уровню ]       │
└──────────────────────────────────────────┘
```

**5 уровней анимации:**

**Уровень 1: ძვალი (10 XP)** — Мелкая радость
- Бомбора mood=`happy`, лёгкое покачивание (усиленный `pb-breath` 0.8s)
- Пара золотых искр (3-4 small divs, `scale(0→1) opacity(1→0)` за 600ms)
- Оверлей закрывается сам через 1.2s

**Уровень 2: ხორცი (30 XP)** — Прыжок радости
- Бомбора mood=`cheer`, 3× bounce (`translateY(0 → -20px → 0)`, 300ms × 3)
- 8 золотых искр в форме взрыва (radial)
- Оверлей закрывается через 1.8s

**Уровень 3: მწვადი (60 XP)** — Танец + конфетти
- Бомбора mood=`cheer`, вращение ±8deg (3 качания × 400ms)
- Конфетти сбоку (переиспользуем `.confetti` из index.css), 8-10 элементов, цвета navy/ruby/gold
- Оверлей закрывается через 2.5s

**Уровень 4: ჩურჩხელა (100 XP)** — Фейерверк
- Бомбора mood=`cheer`, Бомбора увеличивается `scale(1 → 1.15 → 1)` с bounce
- Золотые частицы: 16 золотых кружочков радиально расходятся (SVG или CSS animation)
- Конфетти (navy/ruby/gold/cream) — 12 элементов
- Оверлей закрывается через 3s

**Уровень 5: სუფრა (200 XP)** — Золотой дождь + корона
- Бомбора mood=`cheer` + корона-оверлей (золотой SVG-Crown поверх головы Бомборы)
- Золотой дождь: 20 золотых точек падают сверху вниз, staggered, duration 2s
- Конфетти — 20 элементов, все цвета палитры
- Финальная вспышка: `background` overlay `rgba(245,184,32,0.12)` fade-in/out
- Оверлей закрывается через 4s (или по тапу)

**Crown SVG (оверлей на Бомбору, уровень 5):**
- Абсолютно позиционирован над головой Бомборы (`top: 0, left: 50%, translateX(-50%)`)
- Простая корона: 3 зубца, gold fill (#F5B820), jewelInk border (1.5px)
- Размер: ~40×28px
- Появляется с анимацией: `scale(0.5) translateY(-10px) → scale(1) translateY(0)`, 350ms bounce

**Закрытие оверлея:**
- Тап в любом месте — немедленное закрытие
- Авто-закрытие по таймеру (по уровням — указано выше)
- После закрытия: Бомбора на Dashboard переходит в mood=`cheer` на 3 секунды

## Технические заметки для Developer

**Новые поля в ProgressState:**
```typescript
xpSpent: number   // новое поле, default 0
// AvailableXp = xp - xpSpent (вычисляемое, не хранится)
```

**API:**
- `POST /api/miniapp/treat` — `{ treatIndex: 0-4 }` → `{ ok: true, xpSpent: number, totalTreatsGiven: number }`
- Бэкенд проверяет: `Xp - XpSpent >= price`. Если нет — 400 Bad Request.
- Бэкенд обновляет: `MiniAppUserProgress.XpSpent += price`, `MiniAppUserProgress.TotalTreatsGiven += 1`
- `GET /api/miniapp/me` — добавить `xpSpent: number`, `totalTreatsGiven: number` в ответ

**Frontend:**
- `progress.xpSpent` читается из `/api/miniapp/me` при инициализации
- `availableXp = progress.xp - progress.xpSpent` — вычисляется везде in-place
- После успешной покупки: обновить `progress.xpSpent += price` локально (оптимистичный update)
- Анимационный оверлей — отдельный portal-компонент, рендерится через `document.body`

**Цены (константы в types.ts):**
```typescript
export const TREAT_PRICES = [10, 30, 60, 100, 200] as const
export const TREAT_GEO_NAMES = ['ძვალი', 'ხორცი', 'მწვადი', 'ჩურჩხელა', 'სუფრა'] as const
```

## Адаптивность (375px)

- Bottom sheet: `max-h-[75dvh] overflow-y-auto` — все 5 строк + заголовок + кнопка вмещаются без скролла (~480px): OK при `max-h-[75dvh]` (~560px на 375×812 viewport)
- TreatItem строка: 60px — достаточно при emoji 28px + двухстрочный текст + XP + кнопка
- Кнопка «+»: 36×36px (≥ 44px touch target + padding) — можно сделать 40×40
- Georgian-названия: используем `font-geo` (Noto Sans Georgian), 16px — читается на 375px
- Кнопка «Покормить» на Dashboard: полная ширина, 52px высота — ОК
- FeedAnimation оверлей: центрирован, Бомбора 160px — помещается даже на 375×667 (iPhone SE)
