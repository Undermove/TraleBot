# §79 Конструктор предложений — педагогический контент (пилот: Послелоги)

**Задача:** OWNER-PRIORITIES §79 — методистский блок  
**Автор:** methodist agent, 2026-05-01  
**Пилотный модуль:** Postpositions (послелоги -ში, -ზე, -თან, -დAN, -მდE, -კEN, ქვEShH, уKAн, წIN)

---

## 1. Оценка прогрессии L1→L5+ по CEFR и принципу i+1

### L1 — один пустой слот (A1) ✅ i+1

Ученик видит 3-словное предложение с одним пропуском и пулом из 4–5 чипов. Два пресетных слова полностью известны. Новое — одна постпозиционная форма.

**i+1 verdict: PASS** — ровно одна новая единица. Ошибиться можно только если не знаешь конкретной формы, что и является целью упражнения. CEFR A1.

### L2 — два пустых слота (A1→A2) ✅ i+1 при правильном подборе пар

Два пропуска в 3-словном предложении (пресет = только подлежащее).

**i+1 verdict: PASS** — при условии, что оба пропуска семантически связаны (постпозиция + известный глагол). Риск: если один слот тестирует незнакомую форму глагола, упражнение становится i+2.

**Рекомендация:** в L2-контенте использовать только глаголы из уже пройденных модулей: ვARH/xAR/ARIs (PresentTense), ვcxovrob/cxovrob/cxovrobS (PresentTense), miDIs/modiS (VoM). CEFR A1–A2.

### L3 — подлежащее + сказуемое заданы, собери дополнения (A2) ✅ i+1

Ученик заполняет 2 средних слота: либо две постпозиции (откуда/куда), либо родительный + позиционный послелог (მაგიდის ქვეშ).

**i+1 verdict: PASS** — к моменту L3 ученик видел все послелоги в теории Postpositions L1–L3. Новое — порядок двух известных форм. Один вызов за раз.

**Условие:** предложения с позиционными послелогами (ქვეშ, უкاн, წIN) требуют прохождения Postpositions L3 теории. CEFR A2.

### L4 — только глагол задан (A2/B1 граница) ✅

Знакомые формы, новое — свободная сборка всего вокруг глагола. CEFR A2–B1.

### L5+ — без пресетов (B1) ✅

Полная генерация предложения. Рекомендуется пометить в UI: «Продвинутый / Без подсказок». CEFR B1.

### Итог

Прогрессия **педагогически корректна**. i+1 соблюдается на каждом шаге. Одна системная рекомендация: в L2 для Postpositions-пилота всегда паровать новую постпозицию с уже знакомым глаголом (не тестировать оба элемента как новые).

## 2. Контент пилота — модуль Postpositions

### Глагольные формы, задействованные в предложениях

| Глагол | Используемые формы | Модуль-источник |
|--------|-------------------|-----------------|
| ყოფნა (быть) | ვარ, ხარ, არის, ვართ | PresentTense L1 |
| ცხოვრება (жить) | ვცხოვრობ, ცხოვრობ, ცხოვრობს, ვცხოვრობთ | PresentTense L3 |
| წასვლა/VoM (идти туда) | მივდივარ, მიდიხარ, მიდის, მივდივართ | VoM L1–L2 |
| მოსვლა/VoM (идти сюда) | მოდის | VoM L1–L2 |

Предложения с VoM-глаголами помечены **[VoM]**. Для учеников без VoM-модуля используйте альтернативы с ვARH / ცხოვRObS.

## 3. Стратегия дистракторов

Для каждого уровня пул должен содержать:

| Тип дистрактора | Пример | Педагогическая функция |
|----------------|--------|------------------------|
| Базовая форма без постпозиции | სახლი | Учит отличать форму с послелогом |
| Неправильная постпозиция | სახლზE вместо სახლshi | Тест понимания значения послелога |
| Правильная постпозиция другого существительного | კARTHAN (у двери) вместо у дома | Тест семантической точности |
| Неправильная форма глагола | ვcxovrob вместо cxovrob (2л. вместо 1л.) | Тест спряжения |

## 4. Совместимость с маршрутом без VoM

Предложения **без [VoM]** подходят для учеников на рекомендованном пути (Алфавит → Числа → Знакомство → Местоимения → Настоящее время).

Для полного A1-пилота без VoM достаточно: **L1-1, L1-2, L1-3, L1-4, L1-6** (L1) + **L2-2, L2-5, L2-6** (L2) + **L3-2, L3-3, L3-4** (L3 — только позиционные послелоги с родительным).
