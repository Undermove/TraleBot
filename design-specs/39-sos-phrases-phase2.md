## Обучающий элемент

**Паттерн:** 5 экстренных фраз вводятся как **memorised formulas** в самом начале обучения — без грамматики, только как блоки. Студент учит их через эмоциональный якорь («пригодится в Тбилиси прямо сейчас»), а не через систему.

**Вопросы дизайна:**

1. **Что тут можно показать на грузинском?** 5 коротких фраз целиком: დამეხმარეთ!, პოლიცია!, სასწრაფო!, არ მესმის, სად ვარ?
2. **Когда наступит reveal-момент?** В модуле SOS (Фаза 4): студент видит эти фразы снова — теперь как часть системы, с контекстом, диалогами и объяснениями. Reveal: «это мои старые знакомые из Знакомства».
3. **Как это создаёт дофамин?** Ощущение реальной готовности («я уже могу сказать это в Тбилиси») + последующее узнавание в Фазе 4 («я знал это с первого урока»).

**Ключевое отличие от обычного урока:** SOS-фразы — это не словарь для расширения речи, это **safety formulas**. Дизайн должен передавать: «запомни как пожарный план эвакуации — один раз, навсегда».

### Компонент: новый тип блока `sos`

**Расширение `TheoryBlockDto`:**

```typescript
// В types.ts — добавить 'sos' к union:
type: 'paragraph' | 'list' | 'example' | 'letters' | 'sos'

// Новые поля для sos-блока:
ge?: string          // грузинская фраза (уже существует)
ru?: string          // перевод (уже существует)
translit?: string    // транслитерация (NEW — только для sos)
```

**Рендер в `TheoryBlock` (`LessonTheory.tsx`):**

```
if (block.type === 'sos') {
  jewel-tile
  ruby tag "ЗАПОМНИ" (верхний левый, аналогично ruby tag "пример" в example-блоке)
  Georgian phrase: font-geo, 32px, font-extrabold, text-ruby, leading-tight
  Translit: 11px, jewelInk-hint, tracking-widest, uppercase
  horizontal divider: 1px, jewelInk/15
  Russian: 16px, jewelInk, font-bold
}
```

**Layout (375px):**
```
┌─────────────────────────────────────────────┐
│ [─── ruby tag: ЗАПОМНИ ───]                 │  ← absolute -top-2 left-4 (как "пример")
│                                             │
│ დამეხმარეთ!                                 │  ← 32px, ruby, pt-2
│                                             │
│ DA-ME-KHA-MA-RET                            │  ← 11px, jewelInk-hint, uppercase, tracking-[0.15em]
│ ──────────────────────────────────────────  │  ← border-t border-jewelInk/15, mt-2 pt-2
│ Помогите!                                   │  ← 16px, jewelInk, font-bold
└─────────────────────────────────────────────┘
```

**Детали компонента:**
- Фон: cream (jewel-tile standard)
- Ruby tag: аналогичен существующему "пример" тегу — `bg-cream px-2 py-0.5 border-[1.5px] border-jewelInk rounded-md`, текст ruby
- Грузинская фраза: `font-geo text-[32px] font-extrabold text-ruby leading-tight` — **ruby** (не navy) — один акцент на один смысл: ruby = важно/срочно
- Транслитерация: `font-sans text-[11px] text-jewelInk/40 uppercase tracking-[0.15em] mt-2` — ненавязчиво, вспомогательный элемент
- Разделитель: `border-t border-dashed border-jewelInk/15 mt-2 pt-2`
- Перевод: `font-sans text-[16px] font-bold text-jewelInk` — крупнее обычного, чтобы русское слово запомнилось
- Padding: `px-4 py-4` + `pt-2` для контента под тегом
- Tap target: карточка `min-h-[80px]` — безопасно ≥ 44px ✓

**Почему ruby для Georgian фразы, а не navy:**
- navy = обучающий контент (основная линия)
- ruby = сигнал повышенной важности (в системе: пройденные метки, акцентные лейблы, экстренный модуль)
- Пользователь визуально кодирует: "эти карточки — другие"

| Элемент | Текст |
