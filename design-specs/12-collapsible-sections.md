# Сворачиваемые секции на дашборде

**Задача:** ROADMAP.md → задача #12
**Статус:** ready

## Обучающий элемент

Каждый заголовок секции содержит грузинское слово — тематическую метку раздела:
- **основы** → `საფუძვლები`
- **грамматика** → `გრამატიკა`
- **лексика по темам** → `ლექსიკა`
- **продвинутое** → `გაღრმავება`
- **мой словарь** → `ლექსიკონი`

**Паттерн:** пользователь сворачивает и разворачивает секции десятки раз — в этот момент глаз каждый раз скользит по грузинскому слову. Слово не объясняется явно, но запоминается через повторяющееся действие.

**Reveal-момент:** в модуле «Лексика по темам» пользователь встретит слово `ლექსიკა` в теории — и узнает его: «это же то, что написано на дашборде». Аналогично `გრამატიკა` при проходе модулей грамматики.

## Экраны и состояния

### Экран: Dashboard — секция развёрнута (default)

**Layout:**
```
[ ── основы  საფუძვლები  ─────────────────  4  ˅ ]
  [ jewel-tile: Алфавит ]
  [ jewel-tile: Числа   ]
  ...
```

**Компонент секции:** кнопка-заголовок (`<button>`) во всю ширину дашборда:
- **Левый блок:** `mn-eyebrow` — русское название секции (11px, uppercase, tracking 0.14em, `#5A4735`)
- **Грузинский лейбл:** `font-geo` / Noto Sans Georgian, 10px, `text-jewelInk-hint` (`#9A8A7A`), semibold — рядом с русским, через `gap-3`
- **Разделитель:** горизонтальная линия `h-px bg-jewelInk/15`, `flex-1` (растягивается между лейблом и счётчиком)
- **Счётчик модулей:** цифра, 11px, semibold, `tabular-nums`, `text-jewelInk-mid`
- **Шеврон:** 12×12 SVG, `text-jewelInk-mid`, rotate-0 когда развёрнуто → rotate(-90deg) когда свёрнуто; переход `transition-transform duration-200 ease-out`

**Tap target заголовка:** минимум 44px высотой (`pt-4 pb-3` = 28px padding + контент ≈ 17px = 45px total ✓)

**Состояние развёрнуто:**
- Шеврон: `rotate(0deg)` — смотрит вниз
- Контент (плитки модулей): видимый, не анимируется при первом рендере

**Состояние свёрнуто:**
- Шеврон: `rotate(-90deg)` — смотрит влево
- Контент: скрыт (`!isUserCollapsed && <section>...</section>`)
- Грузинский лейбл в заголовке остаётся виден — это ключевой момент! Свёрнутая секция продолжает показывать грузинское слово

### Экран: Dashboard — секция свёрнута

**Layout:**
```
[ ── основы  საფუძვლები  ─────────────────  4  › ]
(модули скрыты, следующая секция сразу после)
```

**Визуальный эффект:** при сворачивании пользователь видит грузинский лейбл без «окружения» из плиток — он становится более заметным.

### Анимации

**Переключение шеврона:**
- CSS `transition-transform duration-200 ease-out` на элементе SVG
- `rotate(0)` ↔ `rotate(-90deg)` — плавный поворот, 200ms

**Появление/скрытие контента:**
- В текущей реализации — instant show/hide без анимации
- Спека допускает это как v1 (простота > эффект). Если нужен эффект — CSS `max-height` с transition:
  ```
  развёрнуто: max-height: 9999px; transition: max-height 300ms ease-out
  свёрнуто: max-height: 0; overflow: hidden; transition: max-height 200ms ease-in
  ```
  Внимание: `max-height` анимация имеет нелинейное ощущение, приоритет — скорость первого рендера, не плавность.

**Tap feedback:**
- `active:opacity-70 transition-opacity` — стандартный Minankari tap feedback

### Компоненты (новые / изменения)

**Без новых компонентов.** Всё реализуется в `Dashboard.tsx` через state + CSS.

**Изменение типографики грузинского лейбла секции (уточнение):**
- Сейчас: `font-geo text-[10px] text-jewelInk-hint font-semibold`
- Должно быть: Noto Sans Georgian, 10px, `#9A8A7A`, weight 600, letter-spacing 0 (не uppercase)
- Класс `font-geo` должен быть определён в Tailwind config как `font-family: 'Noto Sans Georgian', sans-serif`

**Поведение при длинных Georgian-словах:**
- Лейбл грузинского слова не обрезается (все слова короткие, ≤ 15 символов)
- Если заголовок не помещается в 375px — `mn-eyebrow` обрезается с `truncate`, Georgian-лейбл остаётся

## Персистентность состояния

**Хранение:** `localStorage.getItem('bombora_collapsed')` — JSON-объект вида `{ "grammar": true, "vocab": false }`

**Поведение:**
- Ключи по `section.key`: `basics`, `grammar`, `vocab`, `advanced`, `myvocab`
- `true` = свёрнуто, `false` или отсутствует = развёрнуто
- При ошибке чтения/записи localStorage — молча игнорировать, не ломать UI

**Дефолтные состояния:**
- Новый пользователь (beginner): `basics` развёрнуто, остальные свёрнуты (через `isBeginner`)
- Intermediate: всё развёрнуто
- После первого ручного переключения: состояние фиксируется в localStorage и перекрывает дефолт

**Мой словарь (`myvocab`):**
- Всегда развёрнут по умолчанию
- Пользователь может свернуть — состояние сохраняется

## Адаптивность

**375px (минимум):**
- Заголовок секции: `mn-eyebrow` (≤ 12 символов) + Georgian-лейбл (≤ 14 символов) + flex-1 разделитель + счётчик + шеврон — всё помещается в одну строку без переноса
- Tap target заголовка: ≥ 44px (обеспечено padding-top/bottom)
- Плитки модулей при развёрнутой секции: `px-5`, занимают всю ширину минус 40px отступов

**Начальная прокрутка:**
- На 375px и 5 секциях дашборд длинный. Пользователи со всеми развёрнутыми секциями будут прокручивать. Сворачивание секций сокращает длину страницы — этому способствует дефолт «начни отсюда» (beginner).
