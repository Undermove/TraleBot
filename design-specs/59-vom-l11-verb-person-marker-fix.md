**Оставшаяся проблема — системная ошибка в формах 3-го лица:**

В `questions11.json` ошибочно используется маркер `-ვ-` в формах 3-го лица единственного и множественного числа. Правило однозначно: **только 1-е лицо (ед. и мн.) получает `-ვ-`**. 2-е и 3-е лицо маркера `-ვ-` не имеет.

```
Парадигма imperfect VoM (глагол «идти», приставка მი-):
  1sg: მი-ვ-დი-ოდ-ი = მივდიოდი   ← -ვ- есть
  2sg: მი-დი-ოდ-ი   = მიდიოდი    ← -ვ- НЕТ
  3sg: მი-დი-ოდ-ა   = მიდიოდა    ← -ვ- НЕТ
  1pl: მი-ვ-დი-ოდ-ით = მივდიოდით  ← -ვ- есть
  2pl: მი-დი-ოდ-ით  = მიდიოდით   ← -ვ- НЕТ
  3pl: მი-დი-ოდ-ნენ = მიდიოდნენ  ← -ვ- НЕТ
```

Та же закономерность для других приставок:
- `შე-`: 3sg = **შე**დიოდა (НЕ ~~შევდიოდა~~)
- `ა-`: 3sg = **ა**დიოდა (НЕ ~~ავდიოდა~~), 3pl = **ა**დიოდნენ (НЕ ~~ავდიოდნენ~~)

Текущий файл содержит как минимум 8 вопросов, где неверные формы стоят в **правильном ответе или вопросе-задании** — ученик заучивает невозможную морфологию.

---

## Обучающий элемент

Не применимо — это исправление фактических ошибок, не новая обучающая механика.

Сами исправления косвенно обучают ключевому правилу: **«-ვ- = сигнал 1-го лица, и только 1-го»**. Ученик, видящий `შედიოდა` в контексте «ის» (3sg), правильно закрепляет: «он» — без `-ვ-`.
