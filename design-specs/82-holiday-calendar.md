# §82 Праздничный календарь — контент для пуш-уведомлений

**Создан:** methodist, 2026-05-11  
**Для:** `HolidayCalendarService`, источник `holiday` в `NotificationDispatcher`  
**Проверить у:** native-reviewer (грамматика, регистр, культурная уместность каждой фразы)

---

## Формат записи

| Поле | Описание |
|------|----------|
| `date_fixed` | Фиксированная дата (MM-DD) или `null` для подвижных |
| `date_formula` | Алгоритм вычисления для подвижных праздников |
| `name_ru` | Русское название |
| `name_ka` | Название по-грузински (груз. графика) |
| `greeting_ka` | Фраза-поздравление (груз. графика) |
| `transliteration` | Практическая транслитерация для русскоязычных |
| `translation_ru` | Перевод фразы |
| `cultural_note` | Культурный контекст, 1–2 предложения |

## Педагогические рекомендации для push-текстов

1. **Фраза + транслитерация + перевод обязательны** во всех праздничных пушах — это делает уведомление доступным для любого уровня (A0–B2).
2. **Используйте неформальный регистр (ты-форма)** для поздравлений одному пользователю: `გილოცავ` (gilocav = «я поздравляю тебя»), не `გილოცავთ` (gilocavt = «мы поздравляем вас»). Исключение: Деда-эна и День независимости — там `გილოцავთ` уместен как общественный праздник.
3. **Не усложнять фразу** выше A1-уровня: ни эргатива, ни превербов, ни версионных гласных в push-тексте. Одно простое приветствие.
4. **Стрик-фразы** уже хорошо подобраны: `შვიდი დღე` (7 days, числа знакомы из модуля Числа). Для 30-дневного стрика: `ოცდაათი დღე` — слово сложное (виджесимальная система), поэтому стоит дать разбивку: `ოცდაათი (20+10) დღე — «тридцать дней»`. Для 100 дней: `ასი დღე` — проще, отлично.
