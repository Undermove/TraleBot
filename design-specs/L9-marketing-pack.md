## Обучающий элемент

**Маркетинговый нарратив: «UI, который сам учит грузинскому»** — главная крючковая фраза.

Каждый скриншот должен демонстрировать, что обучение встроено в интерфейс, а не добавлено поверх:
- Лоадер = буква **ქ** (первая буква слова ქართული)
- Нумерация модулей = грузинские числительные **ა ბ გ დ ე ვ**
- День недели = **ორშაბათი** (понедельник) в приветствии Бомборы
- Штамп достижения = **კარგია!** (отлично!) на грузинском

**Reveal-момент для маркетинга:** потенциальный пользователь видит скриншоты и понимает — приложение не просто показывает карточки, а превращает навигацию в урок. Это неожиданно и цепляет.

**Телеграм Stars (ვარსკვლავი):** при упоминании Telegram Stars в copy вставляем «ვარსკვლავი» (грузинский «звезда») — маленький образовательный момент прямо в маркетинге.

## Экраны и состояния

### Экран 1 — Dashboard (главный скриншот)

**Цель:** показать что это за приложение и как выглядит «дизайн учит».

**Состояние для скриншота:**
- Бомбора mood=happy, DayOfWeekChip с **ოთხშაბათი** (среда, 9 букв — демонстрирует Georgia alphabet)
- Раздел «старт»: все 5 launch-тайлов видны (Алфавит — InProgress с прогрессом 3/7, остальные = Default)
- LaunchPathBar: первый узел **ა** = navy (done), второй **ბ** = gold (current), остальные серые
- Coming Soon раздел: свёрнут, только заголовок-строка «скоро · მალე (16) ›»
- «Мой словарь» тайл виден под LaunchPathBar

**Визуальные акценты на скриншоте:**
- KilimProgress на Алфавит-тайле показывает частичный прогресс (иллюстрирует геймплей)
- LaunchPathBar центрирован, легко читается — главная новинка для маркетинга

---

### Экран 2 — Lesson Theory (обучающий момент)

**Цель:** показать, как выглядит процесс обучения. Должен выглядеть просто и красиво.

**Состояние для скриншота:**
- Модуль «Алфавит», Урок 2: буква **ბ** как большой jewel-tile
- Под буквой: транслитерация «ba», произношение «как русское Б»
- KilimProgress вверху: 2 из 5 треугольников заполнено (navy)
- Бомбора mood=guide в левом нижнем углу (маленькая, не перекрывает контент)
- Header с золотой надписью «Алфавит · ა»

**Визуальные акценты:**
- Чёткая типографика Manrope — буква занимает ~30% экрана, выглядит как ювелирная карточка
- Kilim-страйп (8px) сверху и снизу подчёркивает грузинскую эстетику

---

### Экран 3 — Practice / Quiz (геймплей)

**Цель:** показать quiz-механику — это не скучный тест, а интерактивный выбор.

**Состояние для скриншота:**
- Вопрос: «Какая это буква?» — показана буква **გ** крупно
- 4 варианта ответа: «ga», «ka», «da», «ba» — в jewel-btn (ни один не выбран, default state)
- Счётчик вопросов: грузинские числительные **③ / ①⓪** → показывает «3/10» по-грузински
- KilimProgress: 2 треугольника из 7 заполнено
- Заголовок: «Алфавит — Урок 2»

**Альтернатива:** показать экран с уже правильным ответом (ответ выделен green / navy) + Бомбора mood=cheer

---

### Экран 4 — Result (достижение)

**Цель:** передать радость завершения урока и грузинскую эстетику штампов.

**Состояние для скриншота:**
- Большой штамп по центру: **«კარგია!»** (отлично!) — Stamp компонент в ruby/gold палитре
- Под штампом: «Урок 2 пройден» + XP +15 ⭐
- Бомбора mood=cheer: прыгает / confetti-анимация
- Кнопки: «Следующий урок» (jewel-btn navy, primary) + «На главную» (ghost)
- KilimProgress: все 5 треугольников заполнены — полоса победно navy

**Ключевой момент:** штамп **«კარგია!»** — самый сильный визуальный элемент на скриншоте. Пользователь видит грузинское слово в момент победы.

---

### Экран 5 — Dashboard: Bombora fed (retention hook)

**Цель:** показать retention-механику (кормление) без объяснения — вызвать любопытство.

**Состояние для скриншота:**
- Бомбора mood=sated: поднимает бокал лимонада, широко улыбается
- Статус-строка под маскотом: «Сытая · ещё 1ч 47мин»
- XP в топбаре: 45 ⭐ (после траты на угощение)
- Кнопка «Покормить · ⭐ XP» задизейблена (серая — Бомбора сыта)

**Почему этот скриншот:** вызывает WTF-реакцию «что это за собачка с лимонадом?» — лучший крючок для соцсетей.

---

### Экран 6 — Pro Paywall (монетизация)

**Цель:** показать честную и прозрачную монетизацию через Telegram Stars.

**Состояние для скриншота:**
- ProPaywall bottom sheet открыт
- Заголовок: «TraleBot Pro»
- Ключевые тезисы: «Один раз — навсегда», «Нет подписок», «Все модули»
- Цена: «150 ★ Telegram Stars» + грузинское слово «ვარსკვლავი = звезда» маленьким текстом под ценой
- Кнопка «Купить за 150 ★» (jewel-btn gold)

### Реферальная ссылка

**Механика (минимальная для запуска):**
- Команда `/share` в боте → бот отвечает:

```
Поделись Бомборой с другом:

👇 Твоя ссылка:
t.me/trale_bot?start=ref_{userId}

Или просто перешли это сообщение 👆
```

**Inline keyboard:**
```
[ Поделиться ]   ← кнопка «Поделиться» с нативным Share API Telegram
```

**Бэкенд (для Developer):**
- При `/start ref_{userId}` — записать в `User.ReferredByUserId`
- Эндпоинт `/api/miniapp/me` добавляет `referralCode: string` (= userId)
- Статистика рефералов — в AdminScreen (не для пользователей)

## Скриншоты: технические требования

**Формат:** снимать в Telegram на реальном устройстве или через DevTools 375×812px.

**Состояния:** снимать после прохождения 2-3 уроков Алфавита, чтобы:
- KilimProgress показывал частичный прогресс (не пустой)
- LaunchPathBar показывал 1 завершённый узел (ა = navy) + 1 текущий (ბ = gold)
- XP ≥ 15 (видно реальное накопление)

**Цветовой режим:** светлая тема (cream background). Не тёмная — Minankari-палитра рассчитана на cream.

**Язык интерфейса:** русский (основной язык). Грузинские элементы — обучающие вкрапления.

**Не снимать:**
- Пустой Dashboard (нет прогресса — не впечатляет)
- Экраны с ошибками или спиннерами
- Экраны с русскими флагами/политически нейтральный контент (страна = «Испания» в примерах)
