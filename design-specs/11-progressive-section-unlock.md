# Прогрессивная разблокировка разделов для новичка

**Задача:** ROADMAP.md → задача #11
**Статус:** ready

## Обучающий элемент

Заблокированные секции показывают Georgian-лейбл раздела **без объяснения**: `გრამატიკა`, `ლექსიკა`, `გაღრმავება`. Пользователь видит незнакомые слова десятки раз, пока проходит предыдущий раздел.

**Reveal-момент:** в уроке «Грамматика» или «Лексика» те же слова встречаются в теории — пользователь узнаёт: «это же название раздела, который я наконец открыл». Слова из заблокированных секций становятся самыми узнаваемыми Georgian-словами в интерфейсе.

**Паттерн «запретный плод»:** заблокированный раздел вызывает любопытство. Georgian-лейбл без перевода усиливает желание разблокировать. Когда пользователь открывает раздел — слово обретает смысл (двойной дофамин: разблокировка + узнавание языка).

## Логика разблокировки (дизайн-решение, не реализация)

Применяется **только** для `userLevel === 'beginner'`. Intermediate видит всё открытым, как сейчас.

Цепочка разблокировки:
```
основы (всегда открыты)
  ↓ все уроки основ пройдены
грамматика (разблокируется)
  ↓ все уроки грамматики пройдены
лексика по темам (разблокируется)
  ↓ все уроки лексики пройдены
продвинутое (разблокируется)

мой словарь → всегда виден (не зависит от цепочки)
```

«Все уроки пройдены» = все lessons в секции есть в `completedLessons` (независимо от процента правильных ответов — критерий разблокировки мягкий).

### Компонент: SectionHeader (заблокированное состояние)

**Locked state:**
```
[ ── основы  საფუძვლები  ─────────────────  🔒 ]
```

- `opacity-60` на весь row
- Lock-иконка вместо шеврона: 12×12 SVG замка, `text-jewelInk-hint`
- Курсор: `cursor-not-allowed` (desktop) / визуально non-tappable (мобайл)
- `pointer-events: none` — не реагирует на тапы
- `aria-disabled="true"`

**Locked state с прогресс-подсказкой:**
Показывать под заблокированным header'ом маленький hint-текст когда предыдущий раздел «почти готов» (≥ 80% пройдено):
```
[ ── грамматика  გრამატიკა  ──────────────  🔒 ]
   Ещё X уроков до разблокировки           ← 11px, text-jewelInk-hint
```

Hint появляется только если предыдущий раздел ≥ 80% done. Иначе locked header без комментариев.

### Анимация разблокировки

Срабатывает **при первом открытии Dashboard** после того, как весь раздел пройден (детектируется через сравнение текущего прогресса с предыдущим известным).

**Sequence (500ms total):**

1. **0ms** — locked header начинает transition: `opacity` `0.6 → 1.0`, длительность 200ms, ease-out
2. **100ms** — lock-иконка исчезает (fade-out 150ms), вместо неё появляется шеврон (fade-in 150ms)
3. **200ms** — gold pulse-ring вокруг header: `box-shadow: 0 0 0 0 rgba(245,184,32,0.6)` → `0 0 12px 4px rgba(245,184,32,0)`, длительность 400ms, ease-out (один импульс, не повторяется)
4. **300ms** — модули анимированно появляются снизу вверх: `translateY(8px) opacity(0)` → `translateY(0) opacity(1)`, stagger 60ms per tile, duration 280ms, ease-out
5. **500ms** — Бомбора меняет mood на `cheer` на 2 секунды (если на дашборде)

CSS-классы:
```css
@keyframes unlock-pulse {
  0%   { box-shadow: 0 0 0 0 rgba(245, 184, 32, 0.6); }
  100% { box-shadow: 0 0 16px 6px rgba(245, 184, 32, 0); }
}
.unlock-pulse {
  animation: unlock-pulse 500ms ease-out forwards;
}

@keyframes tile-appear {
  0%   { opacity: 0; transform: translateY(8px); }
  100% { opacity: 1; transform: translateY(0); }
}
.tile-appear {
  animation: tile-appear 280ms ease-out both;
}
```

**Требование к UX:** анимация происходит один раз (при первом открытии). Сохранять в localStorage ключ `bombora_unlocked_once` — Set разблокированных секций. Если секция уже в Set — без анимации (тихо открыта).

### Компоненты (новые)

#### `LockedSectionHint` (inline-компонент, не отдельный файл)

**Назначение:** маленький текстовый hint под заблокированным header.

**Props (концептуально):**
```typescript
// Inline props:
remainingLessons: number  // сколько уроков осталось до unlock
show: boolean             // показывать только когда ≥ 80% done
```

**Внешний вид:**
```
  Ещё 3 урока до разблокировки
```
- `font-sans text-[11px] text-jewelInk-hint` (цвет `#9A8A7A`)
- `px-5 pb-2` — отступы соответствуют разделу
- Без иконок, без цвета — максимально тихий

**Грузинская вариация:** если осталось ровно 1 урок — показать `ერთი გაკვეთილი` (один урок) вместо «Ещё 1 урок» — маленький easter egg для тех кто дошёл до конца раздела.

## Копирайтинг

| Элемент | Текст |
|---|---|
| Hint ≥ 2 уроков | `Ещё {N} уроков до разблокировки` |
| Hint = 1 урок | `ერთი გაკვეთილი — и раздел откроется` |
| Hint = 1 урок (если пользователь ещё не видел грузинских чисел) | `Ещё 1 урок до разблокировки` |
| Aria-label замка | `Раздел заблокирован. Пройдите предыдущий раздел, чтобы открыть.` |
| (нет отдельного попапа или тоста) | — |

**Ограничения copy:**
- Не использовать «награда», «ачивка», «уровень» — это не геймификация, это учебный путь
- Не писать «вы должны пройти X» — только «ещё X уроков», ориентация на позитив
- Грузинское easter-egg слово `ერთი` — только в hint, не в заголовке раздела

---

## Адаптивность (375px)

- Locked header: на 375px строка `[──основы──]` идентична открытой, только иконка другая → не влияет на layout
- Hint-текст: 11px, `px-5` — помещается в 335px без переноса (`Ещё 3 урока до разблокировки` = ~26 символов × ~6px = ~156px)
- Lock-иконка 12×12 не меняет высоту header (tap target не нужен для locked — `pointer-events: none`)
- Анимация разблокировки: `translateY(8px)` безопасна на любой ширине, не вызывает горизонтального overflow
