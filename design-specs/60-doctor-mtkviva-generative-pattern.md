# «У врача»: реструктурировать «X მtKиВА» как генеративный паттерн

**Задача:** ROADMAP.md → задача #60 · GitHub issue #387  
**Статус:** ready

---

## Контекст проблемы

Модуль «У врача» подаёт конструкцию боли как словарный список замороженных фраз:

| Фраза | Перевод |
|-------|---------|
| თავი მtKиВА | голова болит |
| მუცელი მtKиВА | живот болит |
| ზურგი მtKиВА | спина болит |
| გული მtKиВА | сердце болит |

Ученик заучивает список — но не видит паттерна. Когда врач спрашивает про незнакомую часть тела, ученик теряется, потому что «ухо болит» не было в списке.

**Ключевой педагогический пробел:** слово მtKиВА (სიტყვა) не объяснено как глагол с субъектом. Грузинский говорит «голова болит-мне» (X + глагол), не «у меня болит голова». Без этого понимания ученик не может построить новое выражение из знакомых частей.

---

## Обучающий элемент

### Инвертированная боль — reveal-момент

**Паттерн грузинского языка:** боль — это субъект. Страдающий — скрытый дательный падеж.

```
X (что болит, именит.) + მtKиВА (болит-мне)
────────────────────────────────────────────
   თავი           + მtKиВА = голова болит (у меня)
   ფეხი           + მtKиВА = нога болит (у меня)
   ↑ любая часть тела!
```

**Reveal-момент:** ученик уже видел инверсные глаголы в модуле «Настоящее время» (მინდა = хочется мне, მიყვარს = люблю буквально). Здесь — тот же механизм: глагол кодирует страдающего. Это не исключение — это система. Когда ученик это осознаёт, все предыдущие примеры выстраиваются в одну логику.

**Обучающий вопрос в theory:** «Почему мы говорим *X мtKиВА*, а не *я болею голова*?» — ответ показывает, что грузинский думает о боли иначе: болит-объект, не болею-я.

## Методистская проверка (обязательно перед dev)

1. **Формы корректны:** X + მtKиВА (в именительном падеже). Примеры:
   - თავი მtKиВА ✓ (голова — именит.)
   - ფეხი მtKиВА ✓ (нога — именит.)
   - ყური მtKიВА ✓ (ухо — именит.)
   - მხარი მtKиВА ✓ (плечо — именит.)
   - კბილი მtKиВА ✓ (зуб — именит.)
2. **გtKиВАТ (2л.)** в примере L5 диалога — корректная форма для обращения врача к пациенту на «вы».
3. **Лингвистический комментарий:** მtKиВА несёт в себе скрытый дательный 1л. (м-приставку в полной парадигме: mтKиВА / gtKиВА / stKиВА). Для этого урока достаточно объяснить «болит-мне», не раскрывая полной парадигмы person agreement — это отдельная тема.
4. **Не вводим** формы 2л./3л. (gtKиВА, stKиВА) — это расширение для отдельного урока. MVP — X + мtKиВА = болит мне.
