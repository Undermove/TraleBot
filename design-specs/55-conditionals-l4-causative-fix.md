# Conditionals L4 cnd4_q15: заменить каузативную форму «დამეგვიანებინა»

**Задача:** ROADMAP.md → задача #55 · GitHub issue #342  
**Статус:** ready

---

## Контекст проблемы

В `GeorgianConditionals/questions4.json`, вопрос `cnd4_q15`, используется каузативная форма глагола:

> «სწრაფად წავედი, რომ არ **დამეგვიანებინა**»  
> ("Быстро ушёл, чтобы не опоздать")

`დამეგვიანებინა` — это каузативная форма («заставить опоздать», синтаксически — «чтобы меня не заставили опоздать»). Каузатив относится к Фазе 6 (`6.6 Каузатив и потенциал`) — теме, которая явно находится после модуля «Условные предложения» (Фаза 5).

На момент прохождения L4 Conditionals ученик:
- Знает: настоящее время, аорист, будущее, имперфект, реальное и нереальное условие, целевое придаточное с `רომ` + კონიუნქტივი
- Не знает: каузативные конструкции, формы III серии

Ученик видит незнакомую морфему `-ებინ-` (маркер каузатива) внутри знакомого контекста целевого придаточного, не может применить ни одного известного правила, и вынужден воспринимать вопрос как чёрный ящик. Это разрушает уверенность и нарушает принцип методического курса.

---

## Обучающий элемент

Не применимо — это исправление вопроса, использующего неподготовленную грамматику. Само по себе удаление каузатива обучает правильно: ученик укрепляет навык распознавания целевого придаточного (`რომ` + კონიუნქტივი) на понятных глагольных формах.

## Что меняется

### Файл: `src/Trale/Lessons/GeorgianConditionals/questions4.json`

**Вопрос:** `cnd4_q15`

**Было:**
```json
{
  "id": "cnd4_q15",
  "lemma": "რომ",
  "question": "Переведи: «სწრაფად წავედი, რომ არ დამეგვიანებინა»",
  "options": [
    "Быстро ушёл, чтобы не опоздать",
    "Если быстро уйду, не опоздаю",
    "Быстро ушёл и не опоздал"
  ],
  "answer_index": 0,
  "explanation": "სწრაფად წავედი — «быстро ушёл». რომ არ დამეგვიანებინა — «чтобы не опоздать».",
  "tags": ["translation"],
  "difficulty": 2
}
```

**Стало:**
```json
{
  "id": "cnd4_q15",
  "lemma": "რომ",
  "question": "Переведи: «ვიყიდე ქართული წიგნი, რომ ვისწავლო»",
  "options": [
    "Купил грузинскую книгу, чтобы учиться",
    "Если куплю грузинскую книгу, буду учиться",
    "Купил грузинскую книгу и учился"
  ],
  "answer_index": 0,
  "explanation": "ვიყიდე — «я купил» (аорист). ქართული წიგნი — «грузинская книга». რომ ვისწავლო — «чтобы учиться» (კონიუნქტივი, 1л.ед.).",
  "tags": ["translation"],
  "difficulty": 2
}
```

**Почему эта замена корректна:**

| Элемент | Форма | Статус в курсе |
|---------|-------|----------------|
| `ვიყიდე` | аорист 1л.ед. от ყიდვა (купить) | ✅ изучен (Фазы 2–4) |
| `ქართული წიგნი` | прилаг. + сущ. | ✅ изучен (Фаза 1–2) |
| `ვისწავლო` | კონიუნქტივი 1л.ед. от სწავლა | ✅ тема этого урока (L4) |
| Конструкция `X, რომ Y` | целевое придаточное | ✅ тема этого урока (L4) |

Ни один элемент замены не выходит за пределы Фаз 1–5.

**Почему был каузатив в оригинале:**  
Предположительно, автор хотел выразить «чтобы не опоздать» — распространённую бытовую ситуацию. В грузинском языке эта идея может передаваться каузативом (`რომ არ დამეგვიანებინა`), но и более простыми конструкциями: `რომ არ დავაგვიანო` (სუბიუნქтivI + преверб) или, как здесь, — просто заменой всей ситуации на другую, столь же естественную.

---

## Альтернативный вариант (если reviewer предпочтёт тему «опоздать»)

Если важно сохранить тему «опоздать», можно использовать форму `დავაგვიანო` (კონიუნქტივი 1л.ед. от გვიანდება, без каузатива). Однако `დავაგვიანო` включает преверб `და-` + корень `გვიან-`, что может быть столь же незнакомо ученику. Замена на нейтральный пример с известными словами (`ვიყიდე / ვისწავლო`) предпочтительна.

## Копирайтинг

**Вопрос (грузинский текст):**  
`«ვიყიდე ქართული წიგნი, რომ ვისწავლო»`

**Правильный ответ:**  
`Купил грузинскую книгу, чтобы учиться`

**Дистракторы:**  
- `Если куплю грузинскую книгу, буду учиться` (путает реальное условие с целевым)  
- `Купил грузинскую книгу и учился` (паратаксис, без целевого смысла)

**Объяснение (explanation):**  
`ვიყიდე — «я купил» (аорист). ქართული წიგნი — «грузинская книга». რომ ვისწავლო — «чтобы учиться» (კონიუნქტივი, 1л.ед.).`

Требования к тексту:
1. Явно назвать `ვისწავლო` как კონიუნქტივი — ученик закрепляет термин этого урока
2. Не упоминать `დამეგვიანებინა` — ссылка на удалённую форму создаёт путаницу
3. Рус. язык, без этнических акцентов
