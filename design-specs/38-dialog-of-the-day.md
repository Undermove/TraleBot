## Обучающий элемент

Каждый день на Dashboard появляется один короткий живой диалог из раздела «Выживание» уровня A0-A1.
Пользователь видит грузинские реплики — и тапом раскрывает перевод каждой по отдельности.

Механика «угадай сам — потом проверь» создаёт мини-момент проверки без давления. Пассивный контакт
с языком происходит каждый день при открытии приложения — без необходимости открывать конкретный урок.

**Reveal-момент:** когда пользователь доходит до модуля «Выживание» (Приветствия, Кафе, Такси),
он узнаёт фразы, которые видел каждое утро. Реакция: «Это же мой диалог!» — эмоциональная
привязка превращает случайный контакт в осмысленное обучение.

**Ротация по дню года:**
```typescript
const dayIndex = Math.floor(Date.now() / (1000 * 60 * 60 * 24)) % DIALOGS.length
```
~20 диалогов покрывают три недели без повторений. Диалог меняется ровно в полночь UTC — пользователь
видит один диалог в течение всего дня.

Реплики раскрываются независимо. Повторный тап скрывает перевод (toggle).

### Состояния карточки

| Состояние | Описание |
|-----------|----------|
| **Default** | Грузинские реплики + hint снизу |
| **Partially revealed** | 1-2 реплики раскрыты, hint скрыт |
| **All revealed** | Все реплики раскрыты, появляется badge «Диалог прочитан» (1 сек, затем исчезает) |
| **Collapsed** | Карточка сворачивается до eyebrow-строки по тапу на иконку ↑ |

## Компонент: DialogOfDayCard

**Файл:** `src/Trale/miniapp-src/src/components/DialogOfDayCard.tsx`

**Props:** нет (данные берутся из статичного массива + `dayIndex`)

**Внутреннее состояние:**

```typescript
const [revealed, setRevealedLines] = useState<Set<number>>(new Set())
const [hintSeen, setHintSeen]     = useState(false)
const [collapsed, setCollapsed]   = useState(false)
const [allDoneShown, setAllDone]  = useState(false)

const dayIndex = Math.floor(Date.now() / (1000 * 60 * 60 * 24)) % DIALOGS.length
const dialog   = DIALOGS[dayIndex]

function toggleLine(i: number) {
  setRevealedLines(prev => {
    const next = new Set(prev)
    next.has(i) ? next.delete(i) : next.add(i)
    return next
  })
  if (!hintSeen) setHintSeen(true)
}
```

**Логика allDone:**
```typescript
useEffect(() => {
  if (revealed.size === dialog.lines.length && !allDoneShown) {
    setAllDone(true)
    setTimeout(() => setAllDone(false), 2000)
  }
}, [revealed])
```

## Верстка компонента

### Внешний контейнер

```
.jewel-tile
  px-5 py-4
  mx-5 mb-4           ← отступы как у suggestion-карточки в Hero
```

Граница: стандартная `jewel-tile` (1.5px jewelInk border + 3px offset shadow + gold hairline).
**Accent: navy** — синяя хейрлайн вместо стандартной gold, чтобы отличать от CTA-карточки suggestion
(которая ruby). Реализуется через `border-navy` + `before:bg-navy` (аналогично существующему
паттерну `border-navy` в `FirstVocabBanner`).

### Eyebrow-строка

```
┌─────────────────────────────────────────────────────────────┐
│ [ДИАЛОГ ДНЯ   navy eyebrow 10px]       [↑ кнопка 20×20]   │
└─────────────────────────────────────────────────────────────┘
```

```tsx
<div className="flex items-center justify-between mb-3">
  <span className="mn-eyebrow text-navy">{dialog.topic.toUpperCase()}</span>
  <button
    onClick={() => setCollapsed(c => !c)}
    className="w-5 h-5 flex items-center justify-center text-jewelInk/40 active:text-jewelInk
               transition-colors p-3 -m-3"
    aria-label={collapsed ? 'Развернуть' : 'Свернуть'}
  >
    <svg width="12" height="12" viewBox="0 0 12 12" fill="none">
      <path d={collapsed ? 'M2 4 L6 8 L10 4' : 'M2 8 L6 4 L10 8'}
            stroke="currentColor" strokeWidth="2" strokeLinecap="round" strokeLinejoin="round"/>
    </svg>
  </button>
</div>
```

### Реплика (DialogLine)

Каждая реплика — отдельный `<button>` на всю ширину:

```
┌──────────────────────────────────────────────────────────┐
│  — გამარჯობა!                               [gold dot] │  ← default
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│  — გამარჯობა!                                           │  ← revealed
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─   │  ← разделитель 1px gold/30
│    Привет!                                              │  ← перевод fade-in
└──────────────────────────────────────────────────────────┘
```

**CSS реплики (default):**
- `w-full text-left py-2 px-0`
- `flex items-center justify-between gap-2`
- `active:opacity-70 transition-opacity`
- Грузинский текст: `font-geo text-[16px] font-bold text-jewelInk leading-snug` (T3-T4)
- Тире «—»: рендерится в UI (`— {line.geo}`), не часть данных

**CSS gold dot-индикатора (пока не раскрыто):**
- `w-1.5 h-1.5 rounded-full bg-gold border border-jewelInk/20 shrink-0`
- Скрывается (`opacity-0`) после раскрытия

**CSS перевода:**
- `font-sans text-[13px] text-jewelInk-mid leading-snug mt-1` (T5)
- Анимация: `anim-fade` (уже есть в index.css), 180ms
- Не показывается в collapsed-режиме

**Разделитель между репликами:**
- `h-px bg-jewelInk/10 my-1` — между строками диалога (не после последней)

### Hint-строка

```
[тапни на фразу, чтобы увидеть перевод]
```

- `font-sans text-[11px] text-jewelInk/40 mt-3 text-center` (T6)
- Видна только пока `!hintSeen`
- Исчезает с `transition-opacity duration-200 opacity-0` сразу после первого тапа

### Badge «всё раскрыто»

Появляется на 2 секунды после раскрытия всех реплик:

```
┌────────────────────────────────┐
│  ✓ Диалог прочитан             │  ← navy bg, cream text, border-radius 99px
└────────────────────────────────┘
```

- `mt-3 py-1 px-3 rounded-full bg-navy text-cream font-sans text-[12px] font-semibold`
- `transition-opacity` fade-in 200ms, fade-out 300ms
- Центрирован: `mx-auto w-fit`

## Анимации

| Элемент | Анимация | Параметры |
|---------|----------|-----------|
| Перевод реплики | `anim-fade` (из index.css) | 180ms ease-out |
| Hint исчезает | `opacity` 1→0 | 200ms ease |
| Badge «прочитан» | opacity 0→1→0 | fade-in 200ms, авто-скрытие через 2000ms |
| Свернуть/развернуть | max-height + opacity | 250ms ease-out |
| Gold dot скрывается | `opacity` 1→0 | 150ms ease |

Нет новых keyframe-анимаций — используется существующий `anim-fade` из `index.css`.

## Адаптивность (375px)

- Самая длинная грузинская реплика в датасете: «ვერ ვპოულობ ჩემს სახლს» (~215px при 16px).
  На 375px с padding 5×2 = 365px ширины — помещается без переноса.
  Если реплика длиннее (теоретически): `leading-snug` + `whitespace-pre-wrap` = корректный перенос.
- Перевод «Не могу найти свой дом» (~195px) — аналогично помещается.
- Максимум 3 реплики: при 16px geo + 13px ru → 2×(16+13+4+1) = ~68px на раскрытую реплику,
  3 реплики ≈ 204px + eyebrow 32px + hint 24px + padding 32px = ~292px → укладывается без скролла.
- Tap target каждой реплики: минимум `py-2` = 32px + font 16px ≈ 44px. Укладывается в норму ≥ 44px.
- На 375px карточка с `mx-5` = ширина 335px — достаточно для текста.
