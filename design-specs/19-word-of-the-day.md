**Задача:** ROADMAP.md → задача #19 · GitHub issue (ссылка на #19 в разделе «Уведомление»)
**Статус:** ready

## Обучающий элемент

Каждое утро в 09:00 МСК бот присылает одно грузинское слово A0–A1 с транслитерацией,
переводом и 2–3 примерами употребления из разговорного контекста.

Механика: **пассивный ежедневный контакт** с конкретным словом. Пользователь видит слово
утром в мессенджере — без необходимости открывать мини-апп. Когда позже встречает это
слово в уроке «Алфавит», «Выживание» или «Числа» — срабатывает **эффект узнавания**
(«это же моё утреннее слово!»). Узнавание без зубрёжки.

**Reveal-момент:** пользователь проходит модуль «Выживание» и видит «მადლობა» в диалоге.
Реакция: «Я это слово уже знаю — бот показывал». Эмоциональная привязка превращает
случайный контакт в осмысленное обучение.

**Обучающий элемент в заголовке:** `სიტყვა დღისა` (слово дня — генитивная конструкция
грузинского) повторяется каждый день → пользователь запоминает эту фразу неосознанно.
Впоследствии он её «узнаёт» в реальном грузинском тексте.

**Чем отличается от #37 «Паттерн дня»:**
- #37 учит **грамматику**: суффиксы, спряжения, падежные схемы — абстрактные паттерны.
- #19 учит **лексику**: конкретное слово + разговорный контекст + произношение.
Два канала, два типа контента — не дублируются.

## Telegram-сообщение: формат

Бот поддерживает MarkdownV2. Сообщение компактное — читается за 10–15 секунд.

### Структура сообщения

```
📚 *СЛОВО ДНЯ*

*მადლობა* \[madloba\]
_спасибо_

• *მადლობა*\! — Спасибо\!
• დიდი *მადლობა* — Большое спасибо
• *მადლობა* ყველაფრისთვის — Спасибо за всё

💡 _Используется отдельно или с «დიდი» для усиления\._
```

**Правила форматирования:**
- Первая строка: `📚 *СЛОВО ДНЯ*` — константный заголовок (иконка 📚 фиксирована, не вращается)
- Грузинское слово: `*слово*` жирным + `\[транслитерация\]` в квадратных скобках
- Перевод: курсив `_перевод_` отдельной строкой
- Два-три примера: bullet `•`, грузинское предложение с выделенным словом жирным + `— перевод`
- Финальная заметка: `💡 _...\._` — одно предложение, только для слов с неочевидным употреблением
- InlineKeyboard: одна кнопка

### Варианты структуры: однозначное vs многозначное слово

**Однозначное слово (стандарт):**
```
📚 *СЛОВО ДНЯ*

*სახლი* \[sakhli\]
_дом_

• ეს ჩვენი *სახლი*\. — Это наш дом\.
• *სახლში* ვართ\. — Мы дома\.
• დიდი *სახლი* — большой дом

```
*(Нет заметки — и без неё ясно)*

**Многозначное слово (с заметкой):**
```
📚 *СЛОВО ДНЯ*

*კარგი* \[k'argi\]
_хорошо / хороший_

• *კარგი*\! — Хорошо\! *(в ответ)*
• *კარგი* კაცი — хороший человек
• ძალიან *კარგი* — очень хорошо

💡 _Как ответ означает «ладно / окей» — не только оценку\._
```

## InlineKeyboard

```
[ В словарь 📖 ]
```

- Одна кнопка типа `url`, открывает мини-апп через Telegram deep link
- URL-шаблон: `https://t.me/tralebot/miniapp` (без startapp — Dashboard)
- **MVP:** кнопка открывает Dashboard мини-аппа. Более точный роутинг (открыть
  конкретный модуль) — после реализации deep link из #37-C (issue #338).
- **Пост-MVP:** если слово принадлежит конкретному модулю — `startapp=module_{moduleId}`,
  иначе — `startapp=vocab` для открытия словаря. Оба варианта требуют фронтенд-роутинга
  из задачи #338.

**Метка кнопки:** «В словарь 📖» — нейтральнее «Практиковать», точнее отражает
лексический характер уведомления.

);

## Бэкенд: scheduled job

**Файл для реализации:** `src/Infrastructure/Telegram/ScheduledJobs/WordOfDayJob.cs`

```
Каждый день в 09:00 МСК:
1. Получить список активных пользователей (боты с установленным chatId)
2. Вычислить индекс слова: dayOfYear % 30
3. Получить word[index] из статичного конфига WordOfDayConfig
4. Сформировать MarkdownV2-сообщение по шаблону
5. Для каждого пользователя: SendMessage(chatId, text, parseMode=MarkdownV2,
   replyMarkup=InlineKeyboard([В словарь 📖, url=miniappUrl]))
```

**Условие отправки:**
- Пользователь должен иметь `ReceiveWordOfDay = true` (новый opt-in флаг, дефолт `true`)
- **Не отправлять** если `MiniAppEnabled = false` в конфигурации сервера

**Архитектурное замечание:** если к моменту реализации `PatternOfDayJob` (#37) уже
существует — `WordOfDayJob` повторяет его структуру с минимальными отличиями.
Возможно вынести общий `DailyNotificationJobBase`.

**Конфигурация расписания:** `appsettings.json`
```json
"WordOfDay": {
  "Enabled": true,
  "SendAtHourMsk": 9,
  "SendAtMinuteMsk": 0
}
```

*Если #37 (PatternOfDay) отправляется в то же время, рассмотреть небольшой сдвиг
(например, WordOfDay в 9:00, PatternOfDay в 9:05) — чтобы пользователи не получали
два сообщения одновременно.*

## Состояния и edge cases

| Сценарий | Поведение |
|----------|-----------|
| Пользователь не подписан на бот | Не получает сообщение |
| `MiniAppEnabled=false` на сервере | Job не запускается вовсе |
| Слово #30, следующий день | Ротация: `dayOfYear % 30` — циклична, день 31 → слово #1 |
| Пользователь хочет отписаться | `/settings` в боте → «Слово дня: вкл / выкл» |
| Кнопка нажата в старом клиенте | Fallback — обычная ссылка на бота |
| PatternOfDay (#37) ещё не реализован | WordOfDay реализуется независимо, JobBase общий |
| Пользователь уже видел это слово 30 дней назад | Ротация повторяет — слово А0, и повторение полезно |

## Копирайтинг

| Элемент | Текст |
|---------|-------|
| Заголовок сообщения | `📚 *СЛОВО ДНЯ*` |
| Кнопка | `В словарь 📖` |
| Опция в настройках | «Слово дня» → вкл / выкл |
| Команда отписки | `/settings` → «Слово дня: вкл/выкл» |

**Правила copy:**
- Без восклицательных знаков в инструкции — только в грузинских примерах (если уместно)
- Грузинское слово всегда жирным в примерах (не в пояснении)
- Транслитерация в квадратных скобках рядом со словом, не в отдельной строке
- Заметка `💡` — максимум одно предложение, только для слов с нюансом (идиома, регистр)
- Часть речи указывается только когда помогает: «наречие» не нужно для «კი», «глагол» нужен для «ვიცი»

## Адаптивность / платформа

Сообщение Telegram — не мини-апп, адаптивность управляется клиентом Telegram.

**Ограничения MarkdownV2:**
- Спецсимволы `( ) . ! - | [ ] { } # + =` и другие должны быть экранированы `\`
- Жирный: `*текст*`, курсив: `_текст_`
- Транслитерация в скобках: `\[madloba\]` (скобки экранируются)
- Нет кастомных цветов — только форматирование Telegram

**Tap target кнопки:** InlineKeyboard-кнопка Telegram всегда ≥ 44px высотой.
