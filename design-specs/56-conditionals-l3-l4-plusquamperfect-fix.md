# Conditionals L3+L4: убрать плюсквамперфект сослагательного без подготовки

**Задача:** ROADMAP.md → задача #56 · GitHub issues #275 (L3) + #343 (L4)  
**Статус:** ready

---

## Контекст проблемы

В квизах Conditionals L3 (cnd3_q05) и L4 (cnd4_q05) используется форма плюсквамперфекта сослагательного наклонения глагола «быть» — `ვყოფილიყავი` / `ყოფილიყავი` — без какого-либо объяснения этой морфологии.

Плюсквамперфект сослагательного — тема Фазы 6 (`6.2 Плюсквамперфект` + `6.3 Сослагательное наклонение`). Ученик Conditionals (Фаза 5) не имеет никаких инструментов для анализа этой формы: она не похожа ни на один паттерн из изученного, не объясняется в теории урока и вводится как «данность» среди вопросов, которые иначе последовательны.

Результат: ученик видит форму, которую не может разобрать → воспринимает урок как произвольный → теряет уверенность в системности грузинской грамматики.

---

## Обучающий элемент

Не применимо — это исправление вопросов, использующих неподготовленную грамматику. Корректное исправление само по себе обучает: вопросы строятся только из материала, который ученик может объяснить на основе известных правил.

## Что меняется

### Файл 1: `src/Trale/Lessons/GeorgianConditionals/questions3.json`

**Вопрос:** `cnd3_q05`  
**Проблема:** `ვყოფილიყავი` — II Сослагательное (плюсквамперфект сослагат.) глагола «быть». Фаза 6, не подготовлено.

**Было:**
```
question: "Переведи: «რომ აქ ვყოფილიყავი, დაგეხმარებოდი»"
options: ["Если бы я был здесь, помог бы тебе", "Если я здесь, помогу", "Когда был здесь, помог"]
answer_index: 0
explanation: "რომ აქ ვყოფილიყავი — «если бы я был здесь». დაგეხმარებოდი — «я бы тебе помог»."
```

**Стало (предлагаемая замена):**
```
question: "Переведи: «რომ დრო მქონდეს, დაგეხმარებოდი»"
options: ["Если бы у меня было время, помог бы тебе", "Если есть время, помогу", "Было время и помог"]
answer_index: 0
explanation: "რომ დრო მქონდეს — «если бы у меня было время» (კავშირებითი от «иметь»). დაგეხმარებოდი — «я бы тебе помог» (კონდიციონალი)."
```

**Обоснование замены:**
- `მქონდეს` = I კავშირებითი (I Subjunctive) глагола «иметь» — уже встречается в cnd3_q03 (`«რომ ფული მქონდეს...»`) и cnd3_q08 (`«რომ დრო მქონდეს...»`)
- `დაგეხმარებოდი` = კონდიციონალი — это именно то, чему учит L3
- Аподосис сохраняется без изменений, протасис заменяется на известную ученику структуру

### Файл 2: `src/Trale/Lessons/GeorgianConditionals/questions4.json`

**Вопрос:** `cnd4_q05`  
**Проблема:** `ყოფილიყავი` — II Сослагательное (плюсквамперфект сослагат., 2л.ед.) глагола «быть». Фаза 6, не подготовлено.

**Было:**
```
question: "Переведи: «გავაკეთე ყველაფერი, რომ კმაყოფილი ყოფილიყავი»"
options: ["Я сделал всё, чтобы ты был доволен", "Если всё сделаю, буду доволен", "Всё сделано и доволен"]
answer_index: 0
explanation: "გავაკეთე ყველაფერი — «я сделал всё». რომ კმაყოფილი ყოფილიყავი — «чтобы ты был доволен» (2 л. ед., сослагательное прош.)."
```

**Стало (предлагаемая замена):**
```
question: "Переведи: «ვსწავლობ, რომ ეს გავიგო»"
options: ["Учусь, чтобы это понять", "Если учусь, пойму", "Учился и понял"]
answer_index: 0
explanation: "ვსწავლობ — «я учусь». რომ ეს გავიგო — «чтобы это понять» (კონიუნქტივი от «понять»)."
```

**Обоснование замены:**
- L4 учит целевым придаточным с `რომ + კონიუნქტივი` — замена полностью в теме урока
- `გავიგო` = I კონიუნქტივი глагола «понять/понимать» (გაგება) — простой, прозрачный паттерн, параллельный cnd4_q03 (`«ვსწავლობ, რომ გამოცდა ჩავაბარო»`)
- Нет VoM-глаголов, нет Phase 6 форм, нет сложной морфологии

> **Примечание разработчику:** точные грузинские формы подлежат методистской проверке перед коммитом. Главный критерий: **ни в вопросе, ни в вариантах ответа не должно быть формы `ყოფილიყავი`, `ვყოფილიყავი` или любого другого плюсквамперфекта сослагательного (II კავშირებითი)**. Если предложенная замена звучит неестественно — заменить на другой глагол с тем же требованием.

## Копирайтинг

Все тексты UI — русский. Грузинские формы в questions/options/explanation — как учебные примеры, не интерфейсный язык.

Ключевые требования к explanation:
1. Чётко указать кириллическую транслитерацию значения (`«если бы у меня было время»`)
2. Назвать грамматическую форму по-русски (`კავშირებითი от «иметь»`)
3. Не упоминать «сослагательное прошедшее» или «плюсквамперфект» — эти термины не вводились и не должны вводиться в этом уроке
