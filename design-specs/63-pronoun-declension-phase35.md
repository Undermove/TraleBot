# Репозиционирование «Склонение местоимений» — Phase 3.5

**Задача:** ROADMAP.md → задача #63 · GitHub issue #422 (отложена до curriculum-спринта)  
**Статус:** ready

---

## Контекст проблемы

Модуль «Склонение местоимений» (`pronoun-declension`) сейчас живёт в Phase 5 («продвинутое»), в одном ряду с Аористом и Будущим временем. Но студент сталкивается с формами «ამ», «მის», «მათ» уже в Phase 3 — при изучении Послелогов (мас-тан, ამ-ში, ჩვენ-ივით). Разрыв:

- Прошёл «Падежи» → увидел, как меняются существительные  
- Сразу «Послелоги» → форма «ამ სახლში» остаётся необъяснённой загадкой («ეს → ამ»?)  
- Только спустя 10+ модулей, в Phase 5, узнаёт, что «ეს» и «ამ» — это один предмет в разных падежах

**Решение:** Перенести модуль в Phase 3.5 — сразу после «Падежей», до «Послелогов». Это закрывает пробел и создаёт reveal-момент.

---

## Обучающий элемент

### Reveal-момент 1: «ამ სახლში» — это не новое слово

Ученик, выучив Падежи, видел: «სახლი → სახლში» (дом → в доме). Он ещё не знает, что «ეს სახლი» → «ამ სახლში» — та же логика. После модуля открытие:

```
ეს სახლი       (это дом — именительный)
ამ სახლში      (в этом доме — дательный/местный)
    ↑
  «ეს» в косвенном падеже = «ამ»
```

Та же система, которую студент только что изучил в Падежах — она работает и для «ეს/ეგ/ის».

### Reveal-момент 2: «მე» не меняется — глагол делает всё

Парадоксальный факт, путавший студентов с PresentTense L2 (задача #40) и VerbClasses L6 (задача #46): у 1-го и 2-го лица форма местоимения одинакова во всех падежах. Падеж виден не в местоимении, а в приставке глагола:

```
«მე» = я (именительный)
«მე» = мне (дательный)  ← та же форма!
«ჩემი» = мой (родительный)

«მე გინდხარ» = я нравлюсь тебе  → «გ-» в глаголе = тебе
«მე მინდა»   = мне хочется      → «მ-» в глаголе = мне
```

Это не исключение из системы — это элегантность: форма местоимения постоянна, вся информация — в глаголе.

### Мост к следующему модулю

После PronounDeclension 3.5 студент открывает «Послелоги» и сразу читает:

- «ამ სახლში» — уже знает, что «ამ» = «ეს» в дательном ✓  
- «მასთან» = «ის» (дат. «მას») + «-თან» (послелог «с/у») ✓  
- «ჩემი სახლი» = «ЧEM-» (gen. от «მე») + «სახლი» ✓

Связь немедленная — знание применяется в следующем уроке.

## Архитектура изменений

### Phase 3.5 — компактная версия (3 урока, переработанная)

Фокус: именительный, дательный, родительный — три самых используемых падежа.  
Эргативный — только для «ის → მAN» (уже известен из Падежей).

| Урок | Тема | Новые слова/формы |
|------|------|--------------------|
| L1   | «ის» и указательные в 4 падежах | ის/მAN/მAS/მISI, ეს/ამAN/ამAS/ამIS, ეგ/მAGAN/მAGAS/მAGIS |
| L2   | Личные местоимения | მე/ჩEMI, შEN/შENI, ის/მIS-, ჩVEN/ჩVENI, TQ'VEN/TQ'VENI, ISIN/მATH |
| L3   | Практика в контексте | «ამ სახლში», «მასთან», «ჩემი», «მისი» в предложениях |

### Phase 5 — расширенная версия (2 дополнительных урока)

Существующие уроки L4 и L5 остаются в Phase 5 как «продвинутый раздел» внутри того же модуля:

| Урок | Тема |
|------|------|
| L4   | Творительный и обстоятельственный падежи |
| L5   | Повторение — местоимения в сложных контекстах |

> **Реализационная заметка:** Developer может оставить один модуль `pronoun-declension` с 5 уроками, убрав его из `advancedIds` и добавив в `grammarIds`. Уроки L4–L5 доступны при входе, но не попадают в рекомендуемый путь.

### Экран: L2 — Личные местоимения («მე» не меняется)

**Теория:**

```
PARAGRAPH:
«У 1-го и 2-го лица местоимения не меняются.
 Падеж виден в приставке глагола, а не в слове.»

TABLE (2-col: форма + значение):
  მე         = я (им.) / мне (дат.) — одна форма для обоих
  შEN        = ты (им.) / тебе (дат.)
  ჩEMI       = мой (род.)
  შENI       = твой (род.)

PARAGRAPH:
«3-е лицо меняется — ის в дательном становится «მAS»:»

TABLE:
  ის → მAN (эрг.) / მAS (дат.) / მISI (род.)
  ისINI → მATH (эрг./дат.) / მATHI (род.)

HIGHLIGHT BOX:
«Парадокс: «მე მINDA» = мне хочется. «მე» выглядит как именительный,
 но функция дательная — форма та же. Запомните: у 1/2 лица форма не меняется.»
```

**Quiz:** 4 вопроса — «Мне нравится» = «მე / ჩEM» (форма в именительном/дательном), «Его» = «მISI», «Им» = «მATH», «Это наш (наш)» = «ჩVENI». Multiple choice.
