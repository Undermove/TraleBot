**Что проверяем:** Вопросы типа `audio-choice` корректно отображаются в PracticeMistakes (разбор ошибок)
**Шаги:**
1. Открыть мини-апп → Числа (или Алфавит) → Урок 5 (или 11) → Практика
2. Намеренно ответить неправильно на 2–3 audio-choice вопроса
3. На экране итогов нажать «разобрать ошибки»
4. В режиме разбора ошибок проверить:
   - Для audio-choice вопросов: должна отображаться карточка «Послушай и выбери» с ▶ кнопкой (не просто текст)
   - Нажать ▶ — должен воспроизводиться OGG файл
   - Выбрать вариант → нажать «проверить»
   - FeedbackBanner: при правильном — показывает грузинский текст (transcript) + перевод; при неправильном — «правильно: {вариант}»
   - Badges вариантов должны быть ა,ბ,გ,დ (не a,b,c,d)
5. Дойти до итога разбора ошибок

**Ожидаемый результат:** Audio-choice вопросы в PracticeMistakes работают идентично Practice.tsx — аудиоплеер присутствует, ответы трекаются, feedback корректный.
